🌱 Position

Position is the vector quantity that specifies the location of a point particle within a defined coordinate system relative to a fixed origin $\mathcal{O}$. Since position describes location in space, it must be represented by a vector $\mathbf{r}$, typically written as a function of time $t$, denoted $\mathbf{r}(t)$. In standard three-dimensional Euclidean space, the position vector is expressed using an orthonormal basis $\{\hat{\mathbf{i}}, \hat{\mathbf{j}}, \hat{\mathbf{k}}\}$ corresponding to the Cartesian axes $(x, y, z)$. Mathematically, the position vector of a particle at time $t$ is defined as:

$$ \mathbf{r}(t) = x(t)\hat{\mathbf{i}} + y(t)\hat{\mathbf{j}} + z(t)\hat{\mathbf{k}} $$

The components $(x(t), y(t), z(t))$ are scalar functions that dictate how the particle's location changes over time. Unlike displacement, which is a change in position ($\Delta \mathbf{r}$), position itself requires specifying *where* the object is relative to $\mathcal{O}$.
