🍃 Constraints

In mathematical optimization, a constraint defines the feasible region—the set of all possible solutions $\mathbf{x}$ that are permissible according to the problem's physical or theoretical limitations. Without constraints, an objective function $\min f(\mathbf{x})$ is often unbounded and ill-posed in any practical context. Constraints fundamentally restrict the domain of variables, transforming a potentially infinite search space into a bounded, manageable feasible set $S$. Mathematically, a standard constrained optimization problem takes the form:

$$ \begin{array}{ll} \text{minimize} & f(\mathbf{x}) \\ \text{subject to} & g_i(\mathbf{x}) \le 0, \quad i=1, \ldots, m \\ & h_j(\mathbf{x}) = 0, \quad j=1, \ldots, k\end{array} $$

Here, $f(\mathbf{x})$ is the objective function, $\mathbf{x} \in \mathbb{R}^n$, and the constraints define the feasible set $S = \{\mathbf{x} \mid g_i(\mathbf{x}) \le 0, h_j(\mathbf{x}) = 0\}$. Inequality constraints ($g_i$) often represent resource limitations (e.g., time or material capacity), while equality constraints ($h_j$) enforce strict relationships between variables (e.g., conservation laws).

Worked Example: Consider maximizing the area $A$ of a rectangular enclosure built using a fixed amount of fencing, say 100 meters. If $\mathbf{x} = \langle l, w \rangle$, the objective function is $f(l, w) = l \cdot w$. The constraints are: first, the perimeter must equal 100m (an equality constraint): $2l + 2w = 100$; second, both dimensions must be non-negative (implicit inequality constraints): $l \ge 0$ and $w \ge 0$. Solving this requires substituting the equality constraint into the objective function or using Lagrangian methods.

Key Theorem: Feasibility. A fundamental requirement for any optimization problem is that the feasible set $S$ must be non-empty; otherwise, no solution exists. If we can find at least one point $\mathbf{x}^* \in S$, the problem is considered feasible. Furthermore, if the objective function $f(\mathbf{x})$ is continuous and the feasible region $S$ is compact (closed and bounded), then by the Extreme Value Theorem, a global minimum and maximum must exist within $S$.

Lab Exercise: Determine whether the following system of constraints has a feasible solution:

$$ \begin{array}{ll} x_1 + 2x_2 \ge 5 \\ -x_1 + x_2 \le 1 \end{array} $$

Graph this system in the $x_1$-$x_2$ plane and identify the corner points of the resulting feasible region. Using these points, estimate the maximum value of the objective function $f(x_1, x_2) = 3x_1 + 4x_2$.
