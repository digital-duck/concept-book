🍃 Hash Table

🍃 Hash Table

A hash table, or hash map, is an abstract data type that implements a dictionary structure by mapping keys to values for efficient retrieval. Fundamentally, it utilizes an array (often called the bucket array) and a specialized function, the hash function $h(k)$, which takes an input key $k$ and computes an integer index within the bounds of the array size $M$. The core objective is to achieve near constant-time complexity for insertion, deletion, and lookup operations. When two distinct keys map to the same index—a situation known as a collision—the hash table must employ a robust collision resolution strategy, such as separate chaining (using linked lists at each bucket) or open addressing (probing for the next available slot). The efficiency of the entire structure hinges on the quality and uniformity of $h(k)$.
