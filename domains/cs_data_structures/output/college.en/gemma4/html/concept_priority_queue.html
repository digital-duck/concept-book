🍃 Priority Queue

🍃 Priority Queue

A priority queue ($\text{PQ}$) is an abstract data type (ADT) similar to a standard queue, but with a crucial modification: each element stored within it has an associated "priority." Unlike a FIFO (First-In, First-Out) queue, where elements are processed strictly in the order they arrived, a $\text{PQ}$ ensures that when an element is dequeued, it is always the one with the highest or lowest priority—depending on whether the implementation is a max-heap or min-heap. This structure maintains elements internally such that access to the maximum (or minimum) priority item is highly efficient. The primary operations are $\text{Insert}(e, p)$, which adds an element $e$ with priority $p$, and $\text{ExtractMax}$ (or $\text{ExtractMin}$), which removes and returns the highest or lowest priority element, respectively.
