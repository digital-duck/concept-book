🍃 Linear Map

A linear map, or linear transformation, is a function between two vector spaces that preserves the operations of vector addition and scalar multiplication. Given two vector spaces $V$ and $W$ over the same field $\mathbb{F}$ (typically $\mathbb{R}$), a mapping $T: V \to W$ is defined as linear if it satisfies two fundamental conditions for all vectors $\mathbf{u}, \mathbf{v} \in V$ and all scalars $c \in \mathbb{F}$:

  • Additivity: $T(\mathbf{u} + \mathbf{v}) = T(\mathbf{u}) + T(\mathbf{v})$
  • Homogeneity (Scalar Property): $T(c\mathbf{u}) = cT(\mathbf{u})$

    • These two properties can be combined into a single, equivalent condition: $T(c_1\mathbf{u} + c_2\mathbf{v}) = c_1T(\mathbf{u}) + c_2T(\mathbf{v})$. Crucially, every linear map must map the zero vector to the zero vector, as $T(\mathbf{0}_V) = T(0 \cdot \mathbf{v}) = 0 \cdot T(\mathbf{v}) = \mathbf{0}_W$.

    Worked Example: Consider $\mathbb{R}^3$ and $\mathbb{R}^2$, both vector spaces over $\mathbb{R}$. Define a function $T: \mathbb{R}^3 \to \mathbb{R}^2$ by $T(x, y, z) = (xy + z, x^2 - y)$. To test for linearity, we must check the scalar property. Let $\mathbf{u} = (1, 0, 0)$ and $c=2$. $T(c\mathbf{u}) = T((2, 0, 0)) = ((2)(0) + 0, (2)^2 - 0) = (0, 4)$. However, $cT(\mathbf{u}) = 2 \cdot T((1, 0, 0)) = 2 \cdot (0, 1) = (0, 2)$. Since $(0, 4) \neq (0, 2)$, the function $T$ is not a linear map. A correct example would be $L(x, y, z) = (ax + by + cz, dx + ey + fz)$, which always satisfies the linearity conditions.

    Key Theorem (Matrix Representation): A fundamental result in linear algebra states that if $V$ and $W$ are finite-dimensional vector spaces over $\mathbb{F}$, then every linear map $T: V \to W$ can be uniquely represented by a matrix multiplication. Specifically, if we choose bases for $V$ and $W$, there exists an $m \times n$ matrix $[A]$ such that $T(\mathbf{v}) = [A] \mathbf{v}$. This theorem allows us to translate the abstract concept of linearity into concrete matrix algebra.
