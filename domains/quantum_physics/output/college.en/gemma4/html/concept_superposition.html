🍃 Superposition

Superposition is a fundamental principle that governs the behavior of linear systems—whether those systems are electrical circuits, differential equations, or physical oscillators. Informally, it states that if multiple sources or inputs act upon a system, the resulting total output is simply the algebraic sum of the outputs that would have been produced by each input acting individually. Mathematically, this principle applies exclusively to linearity; any system characterized by linear behavior adheres to superposition. For a general system $S$ subjected to a set of independent input signals $\{x_1(t), x_2(t), \dots, x_n(t)\}$, the output response $y(t)$ is given by:

$$ y(t) = \sum_{i=1}^{n} y_i(t) $$

where $y_i(t)$ is the response of the system to the input $x_i(t)$. This principle requires the governing differential equations (or transfer function representation) to be linear, meaning they satisfy both additivity and homogeneity.

Consider a simple series DC circuit containing two independent voltage sources, $V_1$ and $V_2$, connected across an unknown load resistor, $R$. To find the total current through $R$, we can apply superposition. We first calculate the current due only to $V_1$ ($I_{V1}$) by setting $V_2=0$, and then calculate the current due only to $V_2$ ($I_{V2}$) by setting $V_1=0$. The total current is simply the sum: $I_{\text{total}} = I_{V1} + I_{V2}$. For example, if $R=6\ \Omega$, $V_1=3\ \text{V}$, and $V_2=9\ \text{V}$, then $I_{V1} = 3/6 = 0.5\ \text{A}$ and $I_{V2} = 9/6 = 1.5\ \text{A}$. Therefore, $I_{\text{total}} = 0.5\ \text{A} + 1.5\ \text{A} = 2.0\ \text{A}$.

The validity of superposition is formally guaranteed by the property that a linear operator $\mathcal{L}$ satisfies:

$$ \mathcal{L}[a x_1(t) + b x_2(t)] = a \cdot \mathcal{L}[x_1(t)] + b \cdot \mathcal{L}[x_2(t)] $$

where $a$ and $b$ are arbitrary constants. This linear combination property is critical for solving complex system responses by decomposing them into simpler, manageable components.

Key Theorem: Linearity of System Response. A physical or mathematical system can be analyzed using the superposition principle if and only if its governing differential equations (or transfer function in the Laplace domain) are linear with respect to the input variables. If non-linear terms (e.g., $V^2$, $\sin(I)$) are present, superposition generally does not apply.
