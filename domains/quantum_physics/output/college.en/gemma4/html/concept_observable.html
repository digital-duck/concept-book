🌱 Observable

An observable is a function that maps elements from an underlying probability space $(\Omega, \mathcal{F}, P)$ to a measurable space $(E, \mathcal{E})$. Intuitively, if $\Omega$ represents all possible outcomes of a random experiment (e.g., the path of a particle), then the observable $X: \Omega \to E$ provides a quantifiable measurement of that outcome—for instance, its position or momentum at a specific time. Formally, an observable must be measurable with respect to $\mathcal{F}$, meaning that for any measurable set $A \in \mathcal{E}$ in the codomain $E$, the preimage $X^{-1}(A) = \{\omega \in \Omega : X(\omega) \in A\}$ must belong to the sigma-algebra $\mathcal{F}$. The measurability condition is crucial because it ensures that the probability of the observable falling into any defined set $A$ can be computed using the underlying measure $P$: $P(X \in A) = P(X^{-1}(A))$.

Consider a simple worked example involving continuous time processes. Suppose $\Omega = [0, T]$ represents the time interval for a Wiener process (Brownian motion), and let the observable $X_T$ be the position of the particle at time $T$. We define $X_T: \Omega \to \mathbb{R}$ by $X_T(\omega) = \omega$. The measurable space is $(\mathbb{R}, \mathcal{B}(\mathbb{R}))$, where $\mathcal{B}(\mathbb{R})$ are the Borel sets. Given that the path measure on $[0, T]$ induces a standard probability distribution on $\mathbb{R}$, $X_T$ is clearly an observable because for any Borel set $A$, $\{ \omega : X_T(\omega) \in A \} = A \cap [0, T]$, which is measurable under the natural sigma-algebra on $[0, T]$.

The key theorem connecting observables and probability theory is the expectation identity. If $X: (\Omega, \mathcal{F}, P) \to (E, \mathcal{E})$ is a real-valued observable such that $\mathbb{E}[|X|] < \infty$, then its expected value can be computed as the integral of the indicator function over its preimage: $\mathbb{E}[X] = \int_{\Omega} X(\omega) dP(\omega)$. This definition allows us to calculate statistical moments (like variance) directly from the underlying probability measure $P$.

Lab Exercise: Given a discrete probability space $\Omega = \{1, 2, 3\}$ with probabilities $P(1)=0.4$, $P(2)=0.3$, and $P(3)=0.3$. Define an observable $Y(\omega) = \omega^2$ representing the squared outcome. Verify that $Y$ is measurable and calculate its expected value $\mathbb{E}[Y]$ using both the definition of expectation for discrete spaces and by treating it as a function integral on the probability space.
