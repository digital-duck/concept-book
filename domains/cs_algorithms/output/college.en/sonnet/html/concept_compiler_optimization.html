🌸 Compiler Optimization

🌸 Compiler Optimization

Definition. Compiler optimization is the systematic transformation of a program's intermediate representation (IR) to produce equivalent code that executes more efficiently—reducing runtime, memory usage, or code size—while preserving observable semantics. Formally, let $P$ denote a program and $\operatorname{llbracket} P \operatorname{rrbracket}$ its denotational semantics. An optimization $\mathcal{O}$ is sound if $\operatorname{llbracket} \mathcal{O}(P) \operatorname{rrbracket} = \operatorname{llbracket} P \operatorname{rrbracket}$ for all inputs. Optimizations are typically organized into passes applied to a control-flow graph (CFG) $G = (V, E)$, where each node $v \in V$ is a basic block and edges represent possible control transfers.

Worked Example: Constant Folding and Dead Code Elimination. Consider the following three-address IR:

$$ \begin{aligned} t_1 &\leftarrow 3 \times 4 \\ t_2 &\leftarrow t_1 + x \\ t_3 &\leftarrow 0 \times y \\ t_4 &\leftarrow t_2 + t_3 \end{aligned} $$

Constant folding evaluates compile-time constants: $t_1 \leftarrow 12$ and $t_3 \leftarrow 0$. Substituting yields $t_4 \leftarrow 12 + x + 0$. Algebraic simplification then reduces this to $t_4 \leftarrow 12 + x$, and $t_3$ becomes unreferenced. Dead code elimination removes any assignment whose result is never subsequently used; here $t_3 \leftarrow 0 \times y$ is removed entirely, eliminating the multiplication. The optimized IR computes the same value in two instructions rather than four.

Key Theorem (Dataflow Convergence). For a monotone dataflow framework $(L, \sqsubseteq, \mathcal{F})$—where $L$ is a finite lattice, $\sqsubseteq$ is the partial order, and $\mathcal{F}$ is a set of monotone transfer functions—the iterative dataflow algorithm converges to the greatest fixed point in at most $|V| \cdot \text{height}(L)$ iterations. This guarantees that analyses such as reaching definitions, live-variable analysis, and available expressions always terminate and yield the most precise solution the framework can express.
