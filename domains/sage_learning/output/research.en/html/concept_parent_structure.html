🌱 Parent Structure

Definition. Let $S = (V, E)$ be a graph, and let $\mathcal{P}$ denote a specific subgraph of $S$. We define the *parent structure* induced by $\mathcal{P}$, denoted $\Pi(\mathcal{P})$, as the minimal spanning forest of the subgraph $S'$ formed by all vertices in $V$ and only those edges $(u, v) \in E$ such that both endpoints $u$ and $v$ are contained within the vertex set $V_{\mathcal{P}}$ corresponding to $\mathcal{P}$. Formally, if $\mathcal{P}$ is connected, then $\Pi(\mathcal{P})$ constitutes a spanning tree of the induced subgraph $S[V_{\mathcal{P}}]$.

Theorem. Given a graph $G=(V, E)$ and any proper subgraph $\mathcal{P} \subset G$ such that the vertex set $V_{\mathcal{P}}$ induces a connected subgraph in $G$, the parent structure $\Pi(\mathcal{P})$ is unique if and only if the induced subgraph $G[V_{\mathcal{P}}]$ is itself a tree.

Proof. ($\Rightarrow$) Assume $\Pi(\mathcal{P})$ is unique. By definition, $\Pi(\mathcal{P})$ is a spanning forest of $G[V_{\mathcal{P}}]$. If $G[V_{\mathcal{P}}]$ contains a cycle, then by standard graph theory results (e.g., related to the Cycle Basis Theorem), there exist multiple sets of edges that span the same vertex set while maintaining acyclicity—i.e., multiple spanning trees are possible. The assumption of uniqueness thus forces $G[V_{\mathcal{P}}]$ to be cycle-free, meaning it must be a tree structure. ($\Leftarrow$) Conversely, if $G[V_{\mathcal{P}}]$ is a tree, then by definition, the unique minimal set of edges required to span all vertices while maintaining connectivity is precisely $E(G[V_{\mathcal{P}}])$ itself; thus $\Pi(\mathcal{P}) = G[V_{\mathcal{P}}]$, which is trivially unique.

Remark. The parent structure concept provides a mechanism for projecting structural information from a complex graph $G$ onto its subcomponents $\mathcal{P}$, effectively isolating the underlying connectivity inherent to $\mathcal{P}$ relative to $G$. This notion generalizes the standard definition of induced subgraphs by explicitly considering only the edges *internal* to $V_{\mathcal{P}}$ and ensuring minimality via spanning forest techniques. Related work in network flow theory utilizes similar concepts when calculating minimum connecting paths within constrained subsets of a larger graph (cf. [1, Section 3.2]). The constraint that $\Pi(\mathcal{P})$ must span $V_{\mathcal{P}}$ makes it particularly useful for analyzing redundancy or critical paths specific to the subgraph structure.
