Definition: An elliptic curve $E$ defined over a finite field $\mathbb{F}_q$ is a smooth projective curve given by a generalized Weierstrass equation, typically written as $y^2 = x^3 + Ax + B$, where $A, B \in \mathbb{F}_q$. The set of non-singular points on $E$, denoted $E(\mathbb{F}_q)$, forms an Abelian group under the chord-tangent method. This structure allows for the computation of the size $|E(\mathbb{F}_q)|$ using algorithms such as point counting or by computing the curve's zeta function $\zeta_E(T)$.

Theorem (Hasse Bound): Let $E$ be an elliptic curve defined over $\mathbb{F}_q$. The number of points on $E$, specifically $|E(\mathbb{F}_q)|$, satisfies the inequality:

$$ | |E(\mathbb{F}_q)| - (q+1) | \le 2\sqrt{q} $$

This bound guarantees that $|E(\mathbb{F}_q)| = q + 1 - t$, where $t$ is the trace of the Frobenius endomorphism, and $t$ must be an integer satisfying $|t| \le 2\sqrt{q}$.
