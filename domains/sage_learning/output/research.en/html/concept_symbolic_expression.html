🍃 Symbolic Expression

A symbolic expression is formally defined as a finite string derived from an alphabet $\Sigma$ which includes variables $V$, constant symbols $C$, and a set of binary or unary operators $\{+, \cdot, f, g, \dots\}$. We denote the set of all well-formed expressions over this alphabet as $\mathcal{E}$. Structurally, these expressions are generally modeled as terms in a free algebra generated by $V$ and $C$. The rigor requires defining an appropriate grammar (e.g., context-free) that governs the permissible arrangement of symbols to ensure semantic validity, thus restricting $\Sigma^*$ to the subset $\mathcal{E} \subset \Sigma^*$.

Theorem 2.1 (Closure Property): Let $(\mathcal{F}, \oplus)$ be a free algebra generated by $V$ and $C$, where $\oplus$ represents an operation defined over two symbolic expressions, $E_1$ and $E_2$. Then the resulting expression $E = E_1 \oplus E_2$ is always contained within the set of valid symbolic expressions, i.e., $E \in \mathcal{E}$. Furthermore, $\mathcal{E}$ forms a monoid under any binary operation $\oplus$ that respects the algebraic axioms (associativity and identity).
