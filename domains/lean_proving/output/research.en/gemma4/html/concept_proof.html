🌱 Proof

Definition. A formal proof $\mathcal{P}$ of a proposition $Q$ within a given axiomatic system $(\mathcal{A}, \mathcal{L})$ is a finite sequence of well-formed formulas (WFFs) $\{F_1, F_2, \ldots, F_k\}$ such that each formula $F_i$ is either an axiom drawn from $\mathcal{A}$, or is derivable from preceding formulas by the application of one or more established inference rules (e.g., Modus Ponens, Generalization). The sequence must conclude with $Q$, and every step in the derivation must be logically justified by the underlying rules of deduction, thereby establishing that $\mathcal{A} \vdash Q$.

Theorem. If a proposition $Q$ is provable within a consistent axiomatic system $\mathcal{A}$, then $Q$ is true (relative to the intended model structure). Formally, consistency implies that $\text{Prov}_{\mathcal{A}}(Q) \implies Q$, where $\text{Prov}_{\mathcal{A}}$ denotes derivability in $\mathcal{A}$.

Proof. The core mechanism establishing this theorem relies on the assumption of consistency. If an axiomatic system $\mathcal{A}$ is consistent, it means that $A \operatorname{land} \operatorname{neg} A$ is not provable for any statement $A$. Gödel's Completeness Theorem (as generalized by Hilbert and Birkhoff) provides the necessary link: for sufficiently expressive formal systems (e.g., first-order logic), derivability ($\vdash$) is equivalent to logical truth ($\models$). If we assume $\mathcal{A} \vdash Q$, this means $Q$ can be derived purely structurally from the axioms. Because the system is consistent, any derivation path that leads to a contradiction must fail; therefore, the resulting proposition $Q$ must hold universally within the intended semantics of the model structure $\mathfrak{M}$.

Remark. The concept of 'proof' highlights the distinction between validity (truth in all models) and provability (derivable within a specific formal system). Systems capable of formalizing arithmetic are subject to limitations, as demonstrated by Gödel's Incompleteness Theorems. These theorems show that if $\mathcal{A}$ is sufficiently powerful ($\text{e.g.}, \mathcal{A} = \text{PA}$) and consistent, there exist true propositions $Q$ such that $\mathcal{A} \not\vdash Q$, thereby necessitating the consideration of meta-mathematical techniques or extensions to address their truth status.
