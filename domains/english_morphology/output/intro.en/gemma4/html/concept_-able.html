🍃 -Able

🍃 -Able

Imagine you are holding a book. If it has pages and lettering, we know something about its quality: someone can read it. The word readable tells us that capability—that this object is able to be read. This suffix, -able, is one of the most useful tools in English because it helps us turn an action or concept into a descriptor, telling us what something is capable of doing. (A suffix is simply a group of letters added to the end of a root word.)

Picture this: The basic idea comes from a verb, like wash. If we just say "wash," it’s an action. But if we add -able, we get washable. This doesn't just mean washing; it means that the item is able to be washed. It shifts the focus from the action itself to the potential for that action.

The simple rule here is: If you can mentally place the phrase "can be" before a word, you might be able to add -able to turn it into an adjective (a describing word). This process tells us about capability. For example, believe $\rightarrow$ believable, because something that generates belief can happen.

Try it yourself! Take the root word comfort. Can we use "can be" before it? What new descriptive word do you get by adding -able to make a word that describes something that is capable of generating comfort?
