🍃 Prefix Root Composition

🍃 Prefix Root Composition

Everyday example. Think about the word unhappy. You already know what happy means — feeling good. But someone added un- to the front, and suddenly the meaning flipped. That tiny addition, un-, is called a prefix (a word-piece that attaches to the front of another word). The word it attaches to, happy, is called the root (the core word that carries the main meaning). Sticking a prefix onto a root is called prefix-root composition.

Picture this. Imagine a root word as a plain white t-shirt. A prefix is a colorful patch you iron onto the chest. The shirt is still a shirt, but the patch changes how people read it. happy → patch on front → unhappy. The root stays intact; the prefix modifies it.

Simple rule. In prefix-root composition, the new word's meaning is roughly:

$$ \text{new word} = \text{prefix meaning} + \text{root meaning} $$

So re- ("again") $+$ write $=$ rewrite ("write again"). The prefix shifts or sharpens what the root already says.

Try it yourself. Take the root connect. What do you get when you add dis- (meaning "apart" or "not")? Write the new word and guess its meaning before checking a dictionary. Then try re- and inter- with the same root. Notice how each prefix steers the meaning in a completely different direction — the root stays the same, but the prefix is the pilot.
