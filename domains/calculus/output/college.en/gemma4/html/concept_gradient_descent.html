🌸 Gradient Descent

Gradient Descent (GD) is a foundational iterative optimization algorithm used to locate the local minima of a differentiable objective function $J(\mathbf{w})$. Given a loss landscape defined by $J(\mathbf{w})$, where $\mathbf{w} \in \mathbb{R}^d$ represents the model parameters, GD aims to minimize this function without requiring explicit inversion of the Hessian matrix. The core mechanism relies on the principle that moving in the direction opposite to the gradient (the steepest ascent) guarantees a local decrease in the function value. Mathematically, the update rule for the parameters $\mathbf{w}$ at iteration $t+1$ is given by:

$$ \mathbf{w}_{t+1} = \mathbf{w}_t - \eta \nabla_{\mathbf{w}} J(\mathbf{w}_t) $$

where $\eta > 0$ is the learning rate, controlling the step size, and $\nabla_{\mathbf{w}} J(\mathbf{w})$ is the gradient vector of the loss function with respect to $\mathbf{w}$.

$$ x_1 = x_0 - \eta (2x_0) $$
