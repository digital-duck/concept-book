Definition 2.1 (The Diagnostic Problem). Given an observable system output sequence $\mathbf{y} = \{y_1, y_2, \dots, y_t\}$ generated by a latent internal state $x(t) \in X$ and possibly corrupted by noise $\epsilon$, the diagnostic problem seeks to infer the most probable underlying causal structure or parameter set $\theta^*$, such that the likelihood of observing $\mathbf{y}$ is maximized. Formally, we seek $\hat{\theta} = \arg\max_{\theta} P(\theta | \mathbf{y}, M)$, where $M$ denotes the system model constraints. This typically involves estimating hidden parameters or identifying deviations from a nominal baseline state $\theta_{nom}$.

Theorem 2.1 (Minimum Description Length Diagnosis). If the diagnostic model assumes distinct, discrete causal hypotheses $\{H_1, H_2, \dots, H_k\}$, then the optimal hypothesis $\hat{H}$ minimizing the total description length $L(\mathbf{y}, H)$ is given by the maximum likelihood estimator approach:

$$ \hat{H} = \arg\max_{H_i} P(H_i | \mathbf{y}) = \arg\max_{H_i} \frac{P(\mathbf{y}|H_i) P(H_i)}{P(\mathbf{y})} $$

Proof. By Bayes' theorem, the posterior probability $P(H_i|\mathbf{y})$ is proportional to the product of the likelihood $P(\mathbf{y}|H_i)$ and the prior $P(H_i)$. Minimizing the description length under the assumption of uniform prior weights $\left(P(H_i) = \text{const}\right)$ reduces the selection problem to maximizing $P(\mathbf{y}|H_i)$.
