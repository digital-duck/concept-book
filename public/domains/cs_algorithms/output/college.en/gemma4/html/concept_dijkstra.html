🍃 Dijkstra

🍃 Dijkstra

Dijkstra's algorithm is a celebrated graph theory algorithm designed to solve the Single-Source Shortest Path Problem (SSSP). Given a weighted, directed or undirected graph $G=(V, E)$, where edge weights $w(u, v)$ are non-negative, and a designated starting vertex $s \in V$, Dijkstra's determines the shortest path distance from $s$ to every other vertex $v \in V$. The algorithm is fundamentally greedy; it iteratively selects the unvisited vertex that currently has the smallest known distance from $s$, thereby ensuring that when a vertex is finalized, the recorded distance is indeed the minimum possible. It utilizes a priority queue structure to efficiently manage and extract the next closest unvisited vertex, maintaining a set of tentative distances $\text{dist}[v]$ for all vertices $v$.

Initialization sets $\text{dist}[A]=0$, $\text{dist}[B]=\infty$, $\text{dist}[C]=\infty$. The algorithm first extracts $A$.
