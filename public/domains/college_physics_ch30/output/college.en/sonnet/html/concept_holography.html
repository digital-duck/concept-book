🌸 Holography

🌸 Holography

Definition. Holography is a technique for recording and reconstructing the complete wavefront of light scattered from an object, capturing both the amplitude and phase information of the electromagnetic wave. Unlike conventional photography, which records only intensity, a hologram encodes the interference pattern between a coherent reference beam and the object beam on a photosensitive medium. When the developed hologram is illuminated by the reference beam alone, it diffracts light to recreate the original wavefront, producing a three-dimensional image. Mathematically, if the object beam has electric field amplitude $E_o e^{i\phi_o}$ and the reference beam has amplitude $E_r e^{i\phi_r}$, the recorded intensity on the film is:

$$ I = |E_o e^{i\phi_o} + E_r e^{i\phi_r}|^2 = E_o^2 + E_r^2 + 2E_o E_r \cos(\phi_o - \phi_r) $$

The cross-term $2E_o E_r \cos(\phi_o - \phi_r)$ encodes the phase difference, which carries the depth information essential for three-dimensional reconstruction.

Worked Example. A transmission hologram is recorded using a helium-neon laser ($\lambda = 632.8\text{ nm}$) with a reference beam incident at $30°$ to the film normal. The object is placed $0.50\text{ m}$ from the film. Determine the fringe spacing on the holographic plate between the reference beam and a point on the object located directly in front of the plate. The path-length difference between the reference beam and the object beam from that point produces a fringe spacing given by:

$$ d = \frac{\lambda}{2\sin(\theta/2)} $$

For $\theta = 30°$: $d = \frac{632.8 \times 10^{-9}}{2\sin(15°)} \approx 1.22\text{ μm}$, well within the resolving capability of holographic emulsions ($\sim 1\text{ μm}$).

Key Theorem. The Gabor reconstruction theorem states that illuminating a developed hologram with the original reference wave generates two diffracted waves: a real image (converging wave) and a virtual image (diverging wave). The virtual image appears to a viewer at the same location as the original object, with full parallax preserved. Coherence length of the source must exceed the maximum path-length difference: $L_c = \lambda^2/\Delta\lambda \operatorname{gg} \Delta\ell_{\max}$.

Lab Exercise. Using a diode-pumped solid-state laser, record a reflection hologram of a small metallic object on a PFG-03M holographic plate. After development, illuminate the plate with white light and observe how the reconstructed image color shifts with viewing angle. Quantitatively measure the angular dispersion and compare it to the grating equation $m\lambda = d\sin\theta$, where $d$ is the fringe spacing determined from the recording geometry.
