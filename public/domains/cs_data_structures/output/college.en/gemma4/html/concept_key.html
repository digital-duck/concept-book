🌱 Key

A key, in an abstract mathematical context, refers to a minimal or defining set of elements necessary to uniquely generate, characterize, or solve a specific system within a larger algebraic structure. Formally, if $S$ is a structured set and $\mathcal{F}$ is the operation governing its components (e.g., addition, multiplication, linear combination), the key $\{k_1, k_2, \ldots, k_n\} \subset S$ is the smallest subset such that every element $s \in S$ can be expressed as a function of $\mathcal{F}$ applied to elements of $\{k_i\}$. The properties of these keys determine the structure's dimensionality and complexity.

Consider the vector space $V = \mathbb{R}^3$. If we want to define this space using a key set, we seek a basis—a minimal set of linearly independent vectors whose span covers all of $V$. Worked Example: Let $A$ be the matrix formed by the columns $\mathbf{v}_1 = (1, 0, 0)^T$, $\mathbf{v}_2 = (0, 1, 0)^T$, and $\mathbf{v}_3 = (0, 0, 1)^T$. The key set is $K = \{\mathbf{v}_1, \mathbf{v}_2, \mathbf{v}_3\}$. Since these vectors are the standard basis, they form a key because any arbitrary vector $(x, y, z)^T$ can be written as a unique linear combination:

$$ (x, y, z)^T = x\mathbf{v}_1 + y\mathbf{v}_2 + z\mathbf{v}_3 $$

If we removed $\mathbf{v}_1$, the resulting set would fail to generate vectors of the form $(a, 0, 0)^T$ where $a \neq 0$. Thus, linear independence and span are critical properties defining a key.

Key Theorem (Basis Existence): Every finite-dimensional vector space $V$ over a field $\mathbb{F}$ possesses at least one basis, which serves as the fundamental key set for $V$. Furthermore, if $B_1 = \{\mathbf{b}_1, \ldots, \mathbf{b}_n\}$ and $B_2 = \{\mathbf{c}_1, \ldots, \mathbf{c}_n\}$ are two bases for $V$, then they span the same vector space, meaning that every element in $B_1$ is a linear combination of elements in $B_2$, and vice versa.
