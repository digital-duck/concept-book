🍃 Binary Search Tree

A Binary Search Tree (BST) is a specialized type of binary tree that adheres to strict ordering properties governing the placement of nodes. Structurally, for any given node $P$, all values in its left subtree must be less than $P$'s value, and all values in its right subtree must be greater than $P$'s value. Formally, if a BST $T$ contains a key set $S$, then for every node $N \in T$:

$$ \text{Key}(L) < \text{Key}(N) < \text{Key}(R) $$

where $\text{Key}(L)$ is the largest element in the left subtree and $\text{Key}(R)$ is the smallest element in the right subtree. This invariant property guarantees that searching for any key $k$ can be performed efficiently by traversing only one branch at each level, thereby maintaining a logarithmic time complexity under ideal conditions.
