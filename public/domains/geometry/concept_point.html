Point

Definition. A point is the most fundamental object in geometry: it has no dimension, no length, no area, and no volume — it is a precise location in space. In the Euclidean plane $\mathbb{R}^2$, a point $P$ is an ordered pair $(x, y)$ where $x, y \in \mathbb{R}$. More generally, a point in $\mathbb{R}^n$ is an ordered $n$-tuple $(x_1, x_2, \ldots, x_n)$. Points are the atomic building blocks from which all geometric figures — lines, curves, surfaces — are constructed.

The distance between two points $P = (x_1, y_1)$ and $Q = (x_2, y_2)$ in $\mathbb{R}^2$ is given by the Euclidean metric:

$$d(P, Q) = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}.$$

This formula extends directly to $\mathbb{R}^n$: $d(P, Q) = \sqrt{\sum_{i=1}^{n}(q_i - p_i)^2}$.

Worked Example. Let $P = (1, 3)$ and $Q = (4, 7)$. Compute the distance $d(P, Q)$.

$$d(P, Q) = \sqrt{(4-1)^2 + (7-3)^2} = \sqrt{9 + 16} = \sqrt{25} = 5.$$

The displacement in the $x$-direction is $3$ and in the $y$-direction is $4$; these form the legs of a right triangle whose hypotenuse — the direct distance between the two points — has length $5$. This connection to the Pythagorean theorem is not coincidental: the Euclidean metric is defined precisely to encode it.

Key Theorem (Triangle Inequality). For any three points $A$, $B$, $C$ in $\mathbb{R}^n$,

$$d(A, C) \leq d(A, B) + d(B, C),$$

with equality if and only if $B$ lies on the line segment $\overline{AC}$. This is the cornerstone of metric geometry: no indirect path can be shorter than the direct one. A proof sketch applies the Cauchy–Schwarz inequality to the displacement vectors $\mathbf{u} = B - A$ and $\mathbf{v} = C - B$, noting that $\|\mathbf{u} + \mathbf{v}\| \leq \|\mathbf{u}\| + \|\mathbf{v}\|$.

Lab Cell (SymPy).

from sympy import Point

P = Point(1, 3)
Q = Point(4, 7)

dist = P.distance(Q)          # returns 5

print(f"Distance : {dist}")

Running this cell confirms $d(P, Q) = 5$, agreeing with the hand calculation above. Students are encouraged to modify the coordinates and verify that the computed distance always satisfies the Triangle Inequality for a third point $R$ of their choosing.
