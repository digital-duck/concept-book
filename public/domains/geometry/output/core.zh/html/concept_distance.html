Distance

在日常生活中，我们不断地衡量"远近"——从家到学校有多远？两座城市之间的距离是多少？这种对"间隔"的直觉，正是数学中距离概念的起点。

定义。 设 $X$ 是一个非空集合。函数 $d: X \times X \to \mathbb{R}$ 若满足以下三条公理，则称 $d$ 为 $X$ 上的一个距离函数（或度量）：

  • 非负性与同一性： $d(x, y) \geq 0$，且 $d(x, y) = 0 \iff x = y$
  • 对称性： $d(x, y) = d(y, x)$
  • 三角不等式： $d(x, z) \leq d(x, y) + d(y, z)$

    • 这三条公理刻画了我们对"距离"最本质的期待：距离不能为负，来回路程一样长，走直线不会比绕路更远。

    例题。 在 $\mathbb{R}^2$ 中，取欧几里得距离 $d(\mathbf{p}, \mathbf{q}) = \sqrt{(p_1 - q_1)^2 + (p_2 - q_2)^2}$。计算 $\mathbf{p} = (1, 2)$ 与 $\mathbf{q} = (4, 6)$ 之间的距离：

    $$d(\mathbf{p}, \mathbf{q}) = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

    为什么三角不等式成立？ 对于欧几里得距离，三角不等式的本质是柯西-施瓦茨不等式。直观地说，连接两点的线段是最短路径——经过任何中间点 $y$ 的折线 $x \to y \to z$，其总长度不可能比直线 $x \to z$ 更短。这不是巧合，而是欧几里得空间内积结构的必然结果。

    值得注意的是，距离的定义并不依赖于坐标系。曼哈顿距离 $d_1(\mathbf{p}, \mathbf{q}) = |p_1 - q_1| + |p_2 - q_2|$ 同样满足这三条公理，只是度量了不同的"远近"。同一个集合上可以定义多种距离，每种距离赋予空间不同的几何结构。

    练习。 验证曼哈顿距离 $d_1$ 对点 $\mathbf{a} = (0,0)$、$\mathbf{b} = (3,4)$、$\mathbf{c} = (1,1)$ 满足三角不等式，即 $d_1(\mathbf{a}, \mathbf{c}) + d_1(\mathbf{c}, \mathbf{b}) \geq d_1(\mathbf{a}, \mathbf{b})$。比较此结果与欧几里得距离下的情况。
