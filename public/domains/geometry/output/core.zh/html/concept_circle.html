Circle

你有没有注意过，往平静的湖面扔一颗石子，水波会以落点为中心向四周均匀扩散？每一圈波纹上的水点到落点的距离都相同——这正是圆的本质。

定义： 在平面上，到定点 $O$（圆心）的距离等于定长 $r$（半径，$r > 0$）的所有点的集合，称为圆。用集合语言表达：

$$\mathcal{C} = \{P \in \mathbb{R}^2 \mid |PO| = r\}$$

若圆心在坐标原点，圆的方程为 $x^2 + y^2 = r^2$；若圆心在 $(a, b)$，则为

$$(x - a)^2 + (y - b)^2 = r^2$$

例题： 判断点 $P(3, 4)$ 是否在圆心为 $O(0, 0)$、半径为 $5$ 的圆上。

计算 $P$ 到 $O$ 的距离：

$$|PO| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$$

因为 $|PO| = r = 5$，所以 $P$ 在圆上。$\square$

这个例子也揭示了圆与勾股定理的深层联系：$3^2 + 4^2 = 5^2$ 既是一组勾股数，也恰好满足圆的方程。

为什么圆的方程是这个形式？ 圆的定义要求"到圆心距离恒为 $r$"。在坐标系中，两点间距离由勾股定理给出：$\sqrt{(x-a)^2 + (y-b)^2} = r$。两边平方，即得标准方程。方程的每一步都直接来自距离的定义——没有额外假设，只有逻辑推导。这说明圆的代数表达与几何定义是完全等价的。

圆还有一个重要的度量性质：周长 $C = 2\pi r$，面积 $A = \pi r^2$。常数 $\pi \approx 3.14159$ 是周长与直径之比，它的存在保证了所有圆在几何上是相似的——大小不同，形状完全一致。
