Distance

Definition. Let $\mathbb{R}^n$ denote $n$-dimensional Euclidean space. Given two points $\mathbf{p} = (p_1, p_2, \ldots, p_n)$ and $\mathbf{q} = (q_1, q_2, \ldots, q_n)$, the Euclidean distance between them is

$$d(\mathbf{p}, \mathbf{q}) = \|\mathbf{p} - \mathbf{q}\| = \sqrt{\sum_{i=1}^{n}(p_i - q_i)^2}.$$

This formula generalises the familiar Pythagorean theorem: in $\mathbb{R}^2$ it reduces to $\sqrt{(p_1-q_1)^2+(p_2-q_2)^2}$, the length of the hypotenuse of the right triangle whose legs run parallel to the coordinate axes. The function $d$ is non-negative, equals zero precisely when $\mathbf{p} = \mathbf{q}$, and is symmetric: $d(\mathbf{p},\mathbf{q}) = d(\mathbf{q},\mathbf{p})$.

Worked Example. Find the Euclidean distance between $A = (1, -2, 4)$ and $B = (5, 2, 1)$ in $\mathbb{R}^3$.

$$d(A, B) = \sqrt{(5-1)^2 + (2-(-2))^2 + (1-4)^2} = \sqrt{16 + 16 + 9} = \sqrt{41} \approx 6.403.$$

Note that the result is independent of the order of the arguments: $d(A,B) = d(B,A)$, consistent with symmetry.

Key Theorem (Triangle Inequality in $\mathbb{R}^n$). For all $\mathbf{p}, \mathbf{q}, \mathbf{r} \in \mathbb{R}^n$,

$$d(\mathbf{p}, \mathbf{r}) \leq d(\mathbf{p}, \mathbf{q}) + d(\mathbf{q}, \mathbf{r}).$$

Proof sketch. Set $\mathbf{u} = \mathbf{q} - \mathbf{p}$ and $\mathbf{v} = \mathbf{r} - \mathbf{q}$, so $\mathbf{r} - \mathbf{p} = \mathbf{u} + \mathbf{v}$. Then $\|\mathbf{u} + \mathbf{v}\| \leq \|\mathbf{u}\| + \|\mathbf{v}\|$ follows directly from the Cauchy–Schwarz inequality $|\mathbf{u} \cdot \mathbf{v}| \leq \|\mathbf{u}\|\|\mathbf{v}\|$. $\square$

This inequality captures the geometric intuition that a straight-line path between two points is never longer than any detour through a third point. In later sections, this property will be seen as one of the defining axioms of the broader class of distance functions known as metrics.

Lab Cell (SymPy).

from sympy import sqrt, Matrix

A = Matrix([1, -2, 4])
B = Matrix([5,  2, 1])

diff = B - A
dist = sqrt(diff.dot(diff))
print(f"d(A, B) = {dist} ≈ {float(dist):.4f}")
# d(A, B) = sqrt(41) ≈ 6.4031

Running this cell confirms the analytic result and illustrates how SymPy preserves exact radical form before converting to a decimal approximation.
