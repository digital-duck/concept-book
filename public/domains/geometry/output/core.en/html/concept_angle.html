Angle

Angle

每当你打开一扇门，门与门框之间就形成了一个张开的程度——这就是角。从时钟的指针到篮球的投篮弧线，角无处不在，它是几何学最基本的度量之一。

定义。 两条共享同一端点的射线所构成的图形称为角（angle）。共享的端点称为顶点（vertex），两条射线称为角的边（sides）。若射线 $\overrightarrow{OA}$ 与 $\overrightarrow{OB}$ 共享顶点 $O$，则所成的角记作 $\angle AOB$。角的大小度量一条边旋转到另一条边所经过的量，常用度（degree）或弧度（radian）表示：

$$360° = 2\pi \text{ rad}, \quad 1° = \frac{\pi}{180} \text{ rad}$$

按大小分类：锐角满足 $0° < \theta < 90°$，直角 $\theta = 90°$，钝角 $90° < \theta < 180°$，平角 $\theta = 180°$。

例题。 时钟在 3:00 时，时针指向 3，分针指向 12。钟面共 12 格，每格对应 $\frac{360°}{12} = 30°$。时针与分针相差 3 格，因此所成角为：

$$\theta = 3 \times 30° = 90°$$

这是一个直角。

为什么弧度制成立？ 在半径为 $r$ 的圆中，弧度定义为弧长与半径之比：$\theta = \frac{s}{r}$。当弧长恰好等于半径时，$\theta = 1$ rad。绕整圆一周的弧长为 $2\pi r$，所以完整旋转对应 $\frac{2\pi r}{r} = 2\pi$ rad。这不是人为约定，而是圆的几何性质的自然结果——弧度制直接将角与弧长联系起来，使得微积分中的公式（如 $\frac{d}{d\theta}\sin\theta = \cos\theta$）不需要额外的转换系数。

练习。 将 $\theta = 150°$ 转换为弧度，并判断它属于哪一类角。
