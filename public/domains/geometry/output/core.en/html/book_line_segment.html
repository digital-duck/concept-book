Concept Book: Line Segment

Payoff

The Natural Endpoint of Euclidean Structure

Definition. A line segment $\overline{AB}$ is the convex hull of two distinct points $A, B \in \mathbb{R}^n$, defined formally as

$$\overline{AB} = \{ (1-t)\,A + t\,B \mid t \in [0,1] \}.$$

It is the simplest one-dimensional compact convex set, bounded by exactly two extremal points, and it is closed: both endpoints belong to the set. Every construction in Euclidean geometry — polygon, polyhedron, triangulation, parametric curve — is ultimately assembled from line segments, making this object not a beginning but a culmination: the first structure rich enough to carry length, direction, convexity, and parametric traversal simultaneously.

Worked Example. Let $A = (1, 2)$ and $B = (5, 6)$ in $\mathbb{R}^2$. The parametric form gives the midpoint at $t = \tfrac{1}{2}$:

$$M = \tfrac{1}{2}(1,2) + \tfrac{1}{2}(5,6) = (3, 4).$$

The Euclidean length is $|AB| = \sqrt{(5-1)^2 + (6-2)^2} = \sqrt{32} = 4\sqrt{2}$, and the unit tangent vector is $\hat{u} = \tfrac{1}{4\sqrt{2}}(4,4) = \tfrac{1}{\sqrt{2}}(1,1)$, encoding direction independently of scale.

Key Theorem (Segment as Metric Geodesic). In $\mathbb{R}^n$ equipped with the Euclidean metric, the line segment $\overline{AB}$ is the unique shortest path between $A$ and $B$. That is, for any rectifiable curve $\gamma:[0,1]\to\mathbb{R}^n$ with $\gamma(0)=A$ and $\gamma(1)=B$,

$$\mathrm{length}(\gamma) \geq |AB|,$$

with equality if and only if $\gamma$ is a monotone reparametrisation of $\overline{AB}$. This geodesic property is what makes the segment the foundational primitive in computational geometry, numerical integration, and mesh generation.

Lab Cell (SymPy).

from sympy import symbols, sqrt, Matrix

t = symbols('t', real=True)
A, B = Matrix([1, 2]), Matrix([5, 6])
seg = (1 - t)*A + t*B          # parametric segment
midpoint = seg.subs(t, 0.5)
length = (B - A).norm()
print(midpoint, length)         # Matrix([3, 4])  4*sqrt(2)
