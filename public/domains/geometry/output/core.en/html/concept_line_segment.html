Line Segment

Definition. A line segment is the convex hull of two distinct points in Euclidean space. Given points $P = (x_1, y_1)$ and $Q = (x_2, y_2)$ in $\mathbb{R}^2$, the line segment $\overline{PQ}$ is the set

$$\overline{PQ} = \{ (1 - t)\,P + t\,Q \mid t \in [0, 1] \}.$$

The parameter $t$ traces the segment continuously from $P$ (at $t = 0$) to $Q$ (at $t = 1$). Each value of $t$ determines a unique interior or boundary point, so the parameterization is injective on $[0,1]$. The length of $\overline{PQ}$ is the Euclidean distance between its endpoints:

$$|\overline{PQ}| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}.$$

This formula extends naturally to $\mathbb{R}^n$ by summing squared coordinate differences across all $n$ dimensions.

Worked Example. Let $P = (1, 2)$ and $Q = (4, 6)$.

  • Length: $|\overline{PQ}| = \sqrt{(4-1)^2 + (6-2)^2} = \sqrt{9 + 16} = \sqrt{25} = 5.$
  • Point at $t = 0.25$:
    • $$\point = (1 - 0.25)(1, 2) + 0.25(4, 6) = (0.75 + 1,\; 1.5 + 1.5) = (1.75, 3).$$

    This confirms that interior points are uniquely recovered by specifying $t \in (0,1)$, and that the parameterization distributes proportionally along the segment — the point at $t = 0.25$ lies exactly one-quarter of the way from $P$ to $Q$.

    Key Theorem (Triangle Inequality). For any three points $A, B, C \in \mathbb{R}^n$,

    $$|\overline{AC}| \;\leq\; |\overline{AB}| + |\overline{BC}|,$$

    with equality if and only if $B$ lies on the segment $\overline{AC}$. Proof sketch: Apply the Cauchy–Schwarz inequality to the vectors $\mathbf{u} = B - A$ and $\mathbf{v} = C - B$, noting that $|\mathbf{u} + \mathbf{v}| \leq |\mathbf{u}| + |\mathbf{v}|$.

    Lab Cell (SymPy).

    from sympy import Point, Segment, Rational

P = Point(1, 2)
Q = Point(4, 6)
seg = Segment(P, Q)

print("Length  :", seg.length)                              # 5
print("At t=0.25:", seg.arbitrary_point(Rational(1, 4)))   # Point2D(7/4, 3)

    Running this cell reproduces each quantity derived above and verifies the parameterization symbolically, bridging geometric intuition with computational confirmation.
