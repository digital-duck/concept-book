Similarity

Definition. Two geometric figures are similar if one can be obtained from the other by a finite composition of isometries (rotations, reflections, translations) and a uniform scaling. Formally, figures $F$ and $F'$ are similar, written $F \sim F'$, if there exists a similarity transformation $T : \mathbb{R}^n \to \mathbb{R}^n$ of the form

$$T(\mathbf{x}) = k R \mathbf{x} + \mathbf{t},$$

where $k > 0$ is the scale factor, $R$ is an orthogonal matrix ($R^T R = I$), and $\mathbf{t} \in \mathbb{R}^n$ is a translation vector. When $k = 1$ the transformation reduces to a rigid motion (congruence); similarity is strictly weaker than congruence.

For triangles, similarity is characterized by angle equality alone: $\triangle ABC \sim \triangle DEF$ if and only if $\angle A = \angle D$, $\angle B = \angle E$, and $\angle C = \angle F$. The corresponding sides are then proportional:

$$\frac{AB}{DE} = \frac{BC}{EF} = \frac{CA}{FD} = k.$$

Worked Example. Suppose $\triangle ABC$ has sides $AB = 6$, $BC = 8$, $CA = 10$, and $\triangle DEF$ has sides $DE = 9$, $EF = 12$, $FD = 15$. Check whether the triangles are similar.

Compute the ratios:

$$\frac{DE}{AB} = \frac{9}{6} = \frac{3}{2}, \quad \frac{EF}{BC} = \frac{12}{8} = \frac{3}{2}, \quad \frac{FD}{CA} = \frac{15}{10} = \frac{3}{2}.$$

All three ratios equal $k = \tfrac{3}{2}$, so $\triangle ABC \sim \triangle DEF$ with scale factor $\tfrac{3}{2}$. Note also that $6^2 + 8^2 = 100 = 10^2$, confirming both triangles are right triangles, which independently guarantees AA similarity.

Key Theorem (AA Criterion). Two triangles are similar if and only if two pairs of corresponding angles are equal. Proof sketch: Given $\angle A = \angle D$ and $\angle B = \angle E$, the angle-sum identity forces $\angle C = \angle F$. Placing $D$ at $A$ and scaling $\triangle DEF$ by $AB/DE$ yields a congruent copy sharing two angles and one side with $\triangle ABC$; the triangles coincide by ASA congruence, establishing proportionality of all sides. $\square$

Lab Cell (SymPy).

from sympy import symbols, Rational, simplify

AB, BC, CA = 6, 8, 10
DE, EF, FD = 9, 12, 15

ratios = [Rational(DE, AB), Rational(EF, BC), Rational(FD, CA)]
print("Ratios:", ratios)                        # [3/2, 3/2, 3/2]
print("Similar?", len(set(ratios)) == 1)        # True
print("Scale factor k =", ratios[0])            # 3/2
