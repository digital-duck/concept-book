Ray

Definition. A *ray* is a subset of a line that begins at a fixed point and extends infinitely in one direction. Formally, given two distinct points $A$ and $B$ in $\mathbb{R}^n$, the ray with *endpoint* (or *origin*) $A$ passing through $B$ is defined as

$$\overrightarrow{AB} = \{ A + t\,(B - A) \mid t \in [0, +\infty) \}.$$

The point $A$ belongs to the ray; no point "behind" $A$ (i.e., corresponding to $t < 0$) does. The ray is thus a half-line closed at $A$. In the plane $\mathbb{R}^2$, a ray is often specified by its endpoint and a direction vector $\mathbf{d} \neq \mathbf{0}$:

$$r(t) = P + t\,\mathbf{d}, \qquad t \geq 0.$$

Worked Example. Let $P = (1, 2)$ and $\mathbf{d} = (3, -1)$ in $\mathbb{R}^2$. Find the point on the ray at parameter $t = 2$, and determine whether $Q = (7, 0)$ lies on the ray.

*Point at $t = 2$:

$$r(2) = (1,2) + 2\,(3,-1) = (1 + 6,\; 2 - 2) = (7, 0).$$

*Membership check for $Q = (7,0)$: Solve $P + t\,\mathbf{d} = Q$:

$$t = \frac{7 - 1}{3} = 2, \qquad t = \frac{0 - 2}{-1} = 2.$$

Both components yield $t = 2 \geq 0$, so $Q \in \overrightarrow{AB}$. $\square$

Key Theorem (Ray–Ray Intersection). Two rays $r_1(t) = P + t\,\mathbf{d}_1$ and $r_2(s) = Q + s\,\mathbf{d}_2$ in $\mathbb{R}^2$ intersect if and only if there exist $t \geq 0$ and $s \geq 0$ satisfying

$$P + t\,\mathbf{d}_1 = Q + s\,\mathbf{d}_2.$$

This $2 \times 2$ linear system has a unique solution when $\mathbf{d}_1$ and $\mathbf{d}_2$ are not parallel (i.e., $\mathbf{d}_1 \times \mathbf{d}_2 \neq 0$, the scalar 2D cross product), and the intersection is valid precisely when both parameters are non-negative.

Lab Cell (SymPy).

from sympy import symbols, Matrix, solve

t, s = symbols('t s', real=True)
P = Matrix([0, 0]);  d1 = Matrix([1, 1])
Q = Matrix([2, 0]);  d2 = Matrix([-1, 1])

eq = P + t*d1 - (Q + s*d2)
sol = solve([eq[0], eq[1]], [t, s])
print(sol)          # {t: 1, s: 1}  → intersection at (1, 1)
assert sol[t] >= 0 and sol[s] >= 0

Running this cell confirms $t = 1 \geq 0$ and $s = 1 \geq 0$, so the two rays meet at $r_1(1) = (1, 1)$.
