Point

Point

Every map begins with a single pin. Before you can draw a route, measure a distance, or outline a region, you need a way to say where. In geometry, that role belongs to the point — the simplest and most fundamental object in the entire subject.

Definition. A point is a location in space that has no size, no shape, and no dimension. It has position and nothing else. We label points with capital letters and represent them as dots on the page, but the dot is only a visual aid — a true point is infinitely small.

Formally, in a coordinate system, a point in the plane is an ordered pair $P = (x, y)$ where $x, y \in \mathbb{R}$. In three-dimensional space, a point is an ordered triple $P = (x, y, z)$.

Worked Example. Consider the point $A = (3, 5)$. This tells us $A$ sits exactly 3 units along the horizontal axis and 5 units along the vertical axis from the origin $O = (0, 0)$. The point $A$ occupies that single location — it contains no length, no area, nothing to measure except its coordinates.

Now consider $B = (3, 5)$ labeled differently. Despite the different letter, $A$ and $B$ are the same point because they share identical coordinates. A point's identity is its position: two points are equal if and only if their corresponding coordinates match. Conversely, $C = (5, 3)$ is a different point from $A$, because order matters in an ordered pair — $(3, 5) \neq (5, 3)$.

Why it's true. Euclid opens the Elements with: "A point is that which has no part." This is not a theorem but an axiom — an irreducible starting agreement. We accept it because every geometric object (lines, planes, solids) can be described as collections of points, yet no geometric object can be decomposed into anything simpler than a point. It is the atom of geometry.

Practice Problem. A point $P$ in the plane has coordinates $(a,\; 2a - 1)$. If $P$ is the same point as $Q = (4, 7)$, find $a$ and verify that both coordinates match.
