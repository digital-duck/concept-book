Area

Definition. Let $R$ be a bounded region in the plane $\mathbb{R}^2$. The area of $R$, denoted $A(R)$, is the limiting value of a sequence of Riemann sums formed by partitioning $R$ into infinitesimally small rectangles. More precisely, if $R$ is described by $a \le x \le b$ and $g_1(x) \le y \le g_2(x)$ for continuous functions $g_1, g_2$, then

$$A(R) = \int_a^b \bigl[g_2(x) - g_1(x)\bigr]\, dx.$$

Area is always non-negative: $A(R) \ge 0$, and $A(R) = 0$ only when $R$ has no interior — for instance, a single point or a curve.

Worked Example. Find the area enclosed between $f(x) = x^2$ and $g(x) = x + 2$.

Step 1 — Find intersections. Set $x^2 = x + 2$, giving $x^2 - x - 2 = 0$, so $(x-2)(x+1) = 0$. The curves meet at $x = -1$ and $x = 2$.

Step 2 — Identify the upper curve. On $[-1, 2]$, test $x = 0$: $g(0) = 2 > f(0) = 0$, so $g(x) = x+2$ lies above $f(x) = x^2$.

Step 3 — Integrate.

$$A = \int_{-1}^{2} \bigl[(x+2) - x^2\bigr]\, dx = \int_{-1}^{2} \bigl(-x^2 + x + 2\bigr)\, dx.$$ $$= \left[-\frac{x^3}{3} + \frac{x^2}{2} + 2x\right]_{-1}^{2} = \left(-\frac{8}{3} + 2 + 4\right) - \left(\frac{1}{3} + \frac{1}{2} - 2\right) = \frac{9}{2}.$$

Key Theorem (Additivity of Area). If $R = R_1 \cup R_2$ with $R_1 \cap R_2$ having no interior, then $A(R) = A(R_1) + A(R_2)$. This additivity property is what allows complex regions to be decomposed into simpler sub-regions and integrated piecewise.

Lab Cell (SymPy).

from sympy import symbols, integrate, solve

x = symbols('x')
f, g = x**2, x + 2
xs = solve(f - g, x)          # [-1, 2]
area = integrate(g - f, (x, xs[0], xs[1]))
print(area)                    # 9/2
