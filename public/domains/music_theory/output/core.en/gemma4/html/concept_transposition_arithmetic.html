🌸 Transposition Arithmetic

Have you ever noticed how swapping just two digits in a number can radically change its meaning? Consider a simple three-digit number like $921$. It represents nine hundred twenty-one. Now, if we simply swap the first and second digits to get $291$, the value has dropped significantly! This process of switching the positions of digits while keeping the digits themselves constant is called transposition, and analyzing how it affects numerical value forms the basis of Transposition Arithmetic.

Definitionally, Transposition Arithmetic refers to calculating the new value of a number after two or more non-adjacent (or adjacent) place values have been swapped. If we consider an original number $N$, and swap the digits at positions $i$ and $j$ (where $i \neq j$), the resulting number, $N'$, is formed by swapping the coefficients corresponding to those place values while keeping all other digits fixed.

To see this in action, let's work through a concrete example. Consider the number $473$. Here, the digit $4$ is in the hundreds place ($10^2$), $7$ is in the tens place ($10^1$), and $3$ is in the units place ($10^0$). If we transpose (swap) the digits $4$ and $3$, the new number formed is $374$. The value has changed dramatically.
