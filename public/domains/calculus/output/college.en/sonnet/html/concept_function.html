🌱 Function

Definition. A function $f$ from a set $A$ to a set $B$, written $f: A \to B$, is a rule that assigns to each element $x \in A$ exactly one element $f(x) \in B$. The set $A$ is called the domain of $f$, and the set $B$ is called the codomain. The range (or image) of $f$ is the subset $\{f(x) \mid x \in A\} \subseteq B$. The defining criterion is well-definedness: for every $x \in A$, the value $f(x)$ is unique. A relation that assigns more than one output to a single input is not a function.

Worked Example. Consider $f: \mathbb{R} \to \mathbb{R}$ defined by $f(x) = x^2 - 3x + 2$. To evaluate $f$ at $x = 4$:

$$ f(4) = (4)^2 - 3(4) + 2 = 16 - 12 + 2 = 6. $$

To find where $f$ equals zero, solve $x^2 - 3x + 2 = 0$. Factoring gives $(x-1)(x-2) = 0$, so $x = 1$ or $x = 2$. These are the zeros of $f$, corresponding to the $x$-intercepts of its graph. Note that the domain is all of $\mathbb{R}$ and the range is $\left[-\tfrac{1}{4}, \infty\right)$, obtained by completing the square: $f(x) = \left(x - \tfrac{3}{2}\right)^2 - \tfrac{1}{4}$.

Key Theorem (Vertical Line Test). A curve in the $xy$-plane is the graph of a function $f: \mathbb{R} \to \mathbb{R}$ if and only if every vertical line $x = c$ intersects the curve in at most one point.

Proof sketch. If $x = c$ intersects the curve at two distinct points $(c, y_1)$ and $(c, y_2)$ with $y_1 \neq y_2$, then $f(c) = y_1$ and $f(c) = y_2$ simultaneously, violating uniqueness of assignment. Conversely, if each vertical line meets the curve at most once, the assignment $x \mapsto f(x)$ is unambiguous wherever defined. $\square$

Lab Exercise. Let $g(x) = \sqrt{4 - x^2}$.

  1. Determine the natural domain of $g$.
  2. Compute $g(0)$, $g(1)$, and $g(2)$.
  3. Identify the range of $g$ and sketch its graph.
  4. Verify using the vertical line test that $g$ defines a function, and explain why the full circle $x^2 + y^2 = 4$ does not.
