🍃 Partial Derivative

The partial derivative is a fundamental tool in multivariable calculus that measures the instantaneous rate of change of a function with respect to one independent variable, while treating all other variables as constants. Given a function $f$ that depends on multiple variables, say $f(x_1, x_2, \ldots, x_n)$, the partial derivative with respect to $x_i$, denoted $\frac{\partial f}{\partial x_i}$ or $f_{x_i}$, quantifies how the output changes when only the input $x_i$ is perturbed by a small amount, keeping all other inputs fixed. Formally, if we consider $f(x, y)$, the partial derivative with respect to $x$ is defined using the limit definition:

$$\frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x+h, y) - f(x, y)}{h}$$

Critically, when computing $\frac{\partial f}{\partial x}$, we must mathematically treat $y$ (and any other variables) as fixed parameters. This distinction is crucial: partial differentiation only considers the change along a coordinate axis.

Worked Example: Consider the function $f(x, y) = 3x^2 \sin(y) + e^{xy}$. To find $\frac{\partial f}{\partial x}$, we differentiate with respect to $x$, treating $\sin(y)$ and $e^{xy}$'s dependence on $x$ (via the exponent) as variables.
