Limit

Definition. Let $f : D \to \mathbb{R}$ be a real-valued function defined on a set $D \subseteq \mathbb{R}$, and let $a$ be an accumulation point of $D$. We say that the limit of $f$ as $x$ approaches $a$ equals $L \in \mathbb{R}$, written

$$\lim_{x \to a} f(x) = L,$$

if for every $\varepsilon > 0$ there exists $\delta > 0$ such that for all $x \in D$ satisfying $0 < |x - a| < \delta$, we have $|f(x) - L| < \varepsilon$. This is the classical $\varepsilon$-$\delta$ definition due to Weierstrass. Crucially, $f$ need not be defined at $a$ itself; the limit describes the function's tendency, not its value.

Worked Example. We verify that $\displaystyle\lim_{x \to 2}(3x - 1) = 5$.

Given $\varepsilon > 0$, we need $\delta > 0$ such that $0 < |x - 2| < \delta$ implies $|(3x-1) - 5| < \varepsilon$. Simplifying the target inequality:

$$|(3x - 1) - 5| = |3x - 6| = 3|x - 2|.$$

We require $3|x - 2| < \varepsilon$, i.e., $|x - 2| < \varepsilon/3$. Choosing $\delta = \varepsilon/3$, we have: if $0 < |x - 2| < \delta$, then $|f(x) - 5| = 3|x - 2| < 3\delta = \varepsilon$. $\square$

Key Theorem (Uniqueness of Limits). If $\displaystyle\lim_{x \to a} f(x) = L$ and $\displaystyle\lim_{x \to a} f(x) = M$, then $L = M$.

Proof sketch. Suppose $L \neq M$. Set $\varepsilon = |L - M|/2 > 0$. By hypothesis, there exist $\delta_1, \delta_2 > 0$ such that $|f(x) - L| < \varepsilon$ and $|f(x) - M| < \varepsilon$ simultaneously for $x$ within $\min(\delta_1, \delta_2)$ of $a$. The triangle inequality then gives $|L - M| \leq |f(x) - L| + |f(x) - M| < 2\varepsilon = |L - M|$, a contradiction.

Lab Exercise. Using the $\varepsilon$-$\delta$ definition, prove that $\displaystyle\lim_{x \to 3} x^2 = 9$. (Hint: factor $|x^2 - 9| = |x+3||x-3|$, then bound $|x+3|$ by restricting $\delta \leq 1$ first, so that $|x - 3| < 1$ implies $|x + 3| < 7$. Conclude by choosing $\delta = \min\!\left(1,\, \varepsilon/7\right)$.)
