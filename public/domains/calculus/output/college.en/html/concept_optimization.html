Optimization

Optimization

Definition. Given a function $f : \mathbb{R}^n \to \mathbb{R}$, an optimization problem seeks a point $\mathbf{x}^ \in \mathcal{D} \subseteq \mathbb{R}^n$ such that $f(\mathbf{x}^)$ is minimized (or maximized) over the feasible domain $\mathcal{D}$. Formally, we write

$$\mathbf{x}^* = \arg\min_{\mathbf{x} \in \mathcal{D}}\; f(\mathbf{x}).$$

When $\mathcal{D} = \mathbb{R}^n$ and $f$ is differentiable, a necessary condition for a local minimum is that the gradient vanishes: $\nabla f(\mathbf{x}^) = \mathbf{0}$. Such a point is called a critical point or stationary point. To distinguish minima from maxima and saddle points, one examines the Hessian matrix $H = \nabla^2 f(\mathbf{x}^)$: if $H$ is positive definite, $\mathbf{x}^*$ is a strict local minimum.

Worked Example. Minimize $f(x) = x^3 - 3x + 2$ over $\mathbb{R}$.

Set $f'(x) = 3x^2 - 3 = 0$, giving $x = \pm 1$. Compute the second derivative $f''(x) = 6x$. At $x = 1$: $f''(1) = 6 > 0$, so $x = 1$ is a local minimum with $f(1) = 0$. At $x = -1$: $f''(-1) = -6 < 0$, so $x = -1$ is a local maximum with $f(-1) = 4$. Since $f(x) \to -\infty$ as $x \to -\infty$, no global minimum exists on $\mathbb{R}$; the global minimum on a closed interval $[-2, 2]$ is $f(1) = 0$.

Key Theorem (Second-Order Sufficient Conditions). Let $f \in C^2(\mathcal{D})$ and suppose $\nabla f(\mathbf{x}^) = \mathbf{0}$. If $\nabla^2 f(\mathbf{x}^)$ is positive definite, then $\mathbf{x}^$ is a strict local minimizer of $f$. Conversely, if $\nabla^2 f(\mathbf{x}^)$ is negative definite, then $\mathbf{x}^$ is a strict local maximizer. If the Hessian is indefinite, $\mathbf{x}^$ is a saddle point.

For constrained optimization — minimize $f(\mathbf{x})$ subject to $g_i(\mathbf{x}) = 0$ — the Karush–Kuhn–Tucker (KKT) conditions generalize the stationarity requirement via Lagrange multipliers $\lambda_i$: $\nabla f(\mathbf{x}^) + \sum_i \lambda_i \nabla g_i(\mathbf{x}^) = \mathbf{0}$.

Lab Exercise. Consider $f(x, y) = x^2 + y^2 - xy - 3x$.

  • Find all critical points by solving $\nabla f = \mathbf{0}$.
  • Classify each critical point using the Hessian.
  • Restrict $f$ to the constraint $x + y = 4$ and use a Lagrange multiplier to find the constrained minimum. Compare with the unconstrained result.
