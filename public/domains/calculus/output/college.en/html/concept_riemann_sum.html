Riemann Sum

Riemann Sum

Definition. Let $f : [a, b] \to \mathbb{R}$ be a bounded function. A partition of $[a, b]$ is a finite set $P = \{x_0, x_1, \ldots, x_n\}$ with $a = x_0 < x_1 < \cdots < x_n = b$. For each subinterval $[x_{i-1}, x_i]$, choose a sample point $c_i \in [x_{i-1}, x_i]$. The Riemann sum of $f$ with respect to $P$ and the choice of sample points is

$$S(f, P) = \sum_{i=1}^{n} f(c_i)\,\Delta x_i, \qquad \Delta x_i = x_i - x_{i-1}.$$

Geometrically, each term $f(c_i)\,\Delta x_i$ is the signed area of a rectangle of width $\Delta x_i$ and height $f(c_i)$. The sum approximates the net area between the graph of $f$ and the $x$-axis. Common choices for $c_i$ yield the left, right, and midpoint Riemann sums.

Worked Example. Approximate $\int_0^1 x^2\,dx$ using a right Riemann sum with $n = 4$ equal subintervals.

Partition: $\Delta x = \tfrac{1}{4}$, so $x_i = \tfrac{i}{4}$ for $i = 0, 1, 2, 3, 4$. Right endpoints: $c_i = x_i$.

$$S = \sum_{i=1}^{4} \left(\frac{i}{4}\right)^2 \cdot \frac{1}{4} = \frac{1}{64}\left(1^2 + 2^2 + 3^2 + 4^2\right) = \frac{1}{64}(1 + 4 + 9 + 16) = \frac{30}{64} = \frac{15}{32} = 0.46875.$$

The exact value is $\tfrac{1}{3} \approx 0.3333$, so the error is $\approx 0.136$. Increasing $n$ reduces this error.

Key Theorem. If $f$ is continuous on $[a, b]$, then for any sequence of partitions whose mesh $\|P\| = \max_i \Delta x_i \to 0$, the Riemann sums converge to the definite integral:

$$\lim_{\|P\| \to 0} S(f, P) = \int_a^b f(x)\,dx.$$

More generally, $f$ is Riemann integrable on $[a, b]$ if and only if it is bounded and continuous almost everywhere (Lebesgue's criterion).
