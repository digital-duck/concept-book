🍃 Riemann Sum

A Riemann sum provides a fundamental method for approximating the definite integral of a function $f(x)$ over a closed interval $[a, b]$. Conceptually, it models the area under the curve by partitioning the interval into $n$ smaller subintervals and constructing rectangles whose height is determined by the function value at a specific sample point within each subinterval. Given an interval $[a, b]$, we partition it using points $x_0, x_1, \dots, x_n$, where $a = x_0 < x_1 < \dots < x_n = b$. The width of the $i$-th rectangle is $\Delta x_i = x_i - x_{i-1}$. If we assume a uniform partition (meaning all subintervals have equal width), then $\Delta x = \frac{b-a}{n}$. The Riemann sum, denoted $R_n$, is therefore defined as the sum of the areas of these $n$ approximating rectangles:

$$R_n = \sum_{i=1}^{n} f(x_i^*) \Delta x_i$$

where $x_i^*$ is a sample point chosen from the $i$-th subinterval $[x_{i-1}, x_i]$. The choice of $x_i^*$ dictates the specific type of approximation (e.g., using the left endpoint means $x_i^* = x_{i-1}$, while using the right endpoint means $x_i^ = x_i$).

Worked Example: Consider approximating the area under the function $f(x) = x^2$ on the interval $[0, 4]$ using $n=4$ equal subintervals. Here, $\Delta x = \frac{4-0}{4} = 1$. We use the right endpoint rule ($x_i^* = x_i$). The partition points are $\{0, 1, 2, 3, 4\}$. The Riemann sum approximation is calculated as:

$$R_4 = \Delta x [f(1) + f(2) + f(3) + f(4)]$$ $$R_4 = 1 [(1)^2 + (2)^2 + (3)^2 + (4)^2] = 1 [1 + 4 + 9 + 16] = 30.$$

The exact area, found via the Fundamental Theorem of Calculus, is $\int_{0}^{4} x^2 \, dx = \left[\frac{x^3}{3}\right]_0^4 = \frac{64}{3} \approx 21.33$. The Riemann sum provides an estimate that, while accurate for large $n$, generally overestimates the true area when using the right endpoint method on increasing functions.
