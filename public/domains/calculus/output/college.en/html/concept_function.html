Function

Function

Definition. A function $f$ from a set $A$ to a set $B$, written $f: A \to B$, is a rule that assigns to each element $x \in A$ exactly one element $f(x) \in B$. The set $A$ is called the domain of $f$, and $B$ is called the codomain. The range (or image) of $f$ is the subset $\{f(x) \mid x \in A\} \subseteq B$. The defining property is uniqueness: if $x = y$ then $f(x) = f(y)$. A relation that assigns more than one output to some input is not a function.

Worked Example. Let $f: \mathbb{R} \to \mathbb{R}$ be defined by $f(x) = x^2 - 3x + 2$. To evaluate $f$ at $x = 4$:

$$f(4) = 4^2 - 3(4) + 2 = 16 - 12 + 2 = 6.$$

To find the range, complete the square: $f(x) = \left(x - \tfrac{3}{2}\right)^2 - \tfrac{1}{4}$. Since a square is always non-negative, $f(x) \geq -\tfrac{1}{4}$ for all $x \in \mathbb{R}$, and every value $y \geq -\tfrac{1}{4}$ is achieved. Thus the range is $\left[-\tfrac{1}{4}, \infty\right)$, which is strictly smaller than the codomain $\mathbb{R}$.

Key Theorem (Composition and Injectivity). Let $f: A \to B$ and $g: B \to C$. Their composition $g \circ f: A \to C$ is defined by $(g \circ f)(x) = g(f(x))$. A function $f$ is injective (one-to-one) if $f(x_1) = f(x_2)$ implies $x_1 = x_2$, and surjective (onto) if every element of $B$ is in the range of $f$. A function that is both injective and surjective is bijective, and bijections admit an inverse function $f^{-1}: B \to A$ satisfying $f^{-1}(f(x)) = x$ for all $x \in A$. Proof that injectivity is preserved under composition follows directly: if $g(f(x_1)) = g(f(x_2))$, injectivity of $g$ gives $f(x_1) = f(x_2)$, then injectivity of $f$ gives $x_1 = x_2$.

Lab Exercise. Let $f: \mathbb{R} \to \mathbb{R}$, $f(x) = 2x + 1$, and $g: \mathbb{R} \to \mathbb{R}$, $g(x) = x^2$.

  • Compute $(g \circ f)(3)$ and $(f \circ g)(3)$. Are they equal?
  • Determine whether $f$ and $g$ are injective and/or surjective, with justification.
  • Find $f^{-1}$ explicitly and verify that $f^{-1}(f(x)) = x$.
