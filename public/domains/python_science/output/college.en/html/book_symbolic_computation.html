Symbol

Definition. A symbol is a primitive, uninterpreted mark drawn from a fixed alphabet $\Sigma$. Formally, given an alphabet $\Sigma = \{s_1, s_2, \ldots, s_n\}$, each element $s_i$ is a symbol: an atomic unit that carries no intrinsic meaning until assigned one by an interpretation function $\mathcal{I} : \Sigma \to \mathcal{D}$, where $\mathcal{D}$ is a domain of discourse. This separation between a symbol and its referent is the foundation of formal logic, programming language theory, and mathematics itself. Symbols are the raw material from which strings, expressions, terms, and formulas are constructed by applying formation rules.

Worked Example. Consider the propositional language with alphabet $\Sigma = \{p, q, \neg, \wedge, \vee, \Rightarrow, (, )\}$. The sequence of symbols $\neg p \wedge q$ is a well-formed formula (wff) because it is built according to the grammar: if $\phi$ and $\psi$ are wffs, then $\neg \phi$ and $\phi \wedge \psi$ are wffs. Here $p$ and $q$ are propositional symbols — syntactic objects with no fixed truth value. Only under an interpretation $\mathcal{I}(p) = \top$, $\mathcal{I}(q) = \bot$ does the formula evaluate to $\mathcal{I}(\neg p \wedge q) = \bot \wedge \bot = \bot$. The same symbol string admits many interpretations, demonstrating that meaning is not in the mark but in the mapping.

Key Theorem (Substitution Lemma). Let $\phi$ be a formula and let $\mathcal{I}$ and $\mathcal{I}'$ be two interpretations that agree on every propositional symbol occurring in $\phi$. Then $\mathcal{I}(\phi) = \mathcal{I}'(\phi)$.

Proof sketch. By structural induction on $\phi$. The base case holds by hypothesis. For $\neg \phi'$: if $\mathcal{I}(\phi') = \mathcal{I}'(\phi')$ by induction, then $\mathcal{I}(\neg \phi') = \neg \mathcal{I}(\phi') = \neg \mathcal{I}'(\phi') = \mathcal{I}'(\neg \phi')$. The connective cases $\wedge, \vee, \Rightarrow$ follow identically. $\square$

This lemma guarantees that the truth value of a formula depends only on the symbols it actually contains, making symbols the irreducible unit of semantic accountability.

Lab Exercise. Let $\Sigma = \{a, b, c, \neg, \wedge, (, )\}$ and let $\mathcal{I}_1(a)=\top, \mathcal{I}_1(b)=\bot, \mathcal{I}_1(c)=\top$ and $\mathcal{I}_2(a)=\bot, \mathcal{I}_2(b)=\top, \mathcal{I}_2(c)=\top$. (1) Write the formula $\phi = \neg a \wedge (b \wedge c)$ and evaluate $\mathcal{I}_1(\phi)$ and $\mathcal{I}_2(\phi)$. (2) Construct a formula $\psi$ using only $a$ and $c$ such that $\mathcal{I}_1(\psi) = \mathcal{I}_2(\psi)$, and verify this with the Substitution Lemma. (3) Enumerate all $2^3 = 8$ interpretations of $\{a, b, c\}$ and identify those under which $\phi$ is true.

Symbolic Computation

Symbolic Computation

Definition. Symbolic computation (also called computer algebra) is the manipulation of mathematical expressions in their exact, unevaluated form, as opposed to numerical computation, which approximates quantities with floating-point arithmetic. A symbolic system represents objects such as polynomials, rational functions, integrals, and matrices as structured data—trees of operators and operands—and applies rewriting rules to transform them without introducing rounding error. Formally, a term algebra over a signature $\Sigma$ is the set $T(\Sigma, X)$ of all well-formed expressions built from function symbols in $\Sigma$ and variables in $X$; symbolic computation operates on elements of $T(\Sigma, X)$ by applying reduction relations $\to_R$ derived from axioms.

Worked Example. Consider simplifying the expression $\dfrac{d}{dx}\!\left(x^3 \sin x\right)$. A numerical evaluator would sample this derivative at specific points. A symbolic system instead applies the product rule exactly:

$$\frac{d}{dx}\!\left(x^3 \sin x\right) = \frac{d}{dx}(x^3)\cdot \sin x + x^3 \cdot \frac{d}{dx}(\sin x) = 3x^2 \sin x + x^3 \cos x.$$

Each step is a pattern-match against a rewrite rule (e.g., $\frac{d}{dx}(f \cdot g) \to f' g + f g'$), producing an identical closed-form result for every query—no floating-point error accumulates.

Key Theorem (Church–Rosser / Confluence). A rewriting system $(\mathcal{T}, \to_R)$ is confluent if, whenever $t \to_R^ u$ and $t \to_R^ v$, there exists $w$ such that $u \to_R^ w$ and $v \to_R^ w$. Confluent, terminating systems are called canonical (or complete): every expression reduces to a unique normal form, guaranteeing that symbolic simplification is deterministic regardless of the order in which rules are applied. The Knuth–Bendix completion algorithm can extend a non-confluent rule set into a canonical one when a suitable term ordering exists.

Payoff

Definition. Symbolic computation is the exact, rule-governed manipulation of mathematical expressions represented as structured syntax trees, as opposed to floating-point evaluation of numerical approximations. A symbolic system operates on elements of a free algebra $\mathcal{F}(\Sigma, X)$—the set of all well-formed terms over an alphabet $\Sigma$ of operators and a set $X$ of indeterminates—and applies rewrite rules of the form $\ell \to r$ that preserve semantic equivalence. Because every transformation is a provable identity rather than a rounded estimate, symbolic computation is the natural endpoint of a curriculum that begins with formal language and culminates in mechanised reasoning: it closes the loop from representation to certified correctness.

Worked example. Consider the definite integral $I = \int_0^1 x^2 \ln x \, dx$. Numerical quadrature yields $I \approx -0.1111$, but carries no certificate. A symbolic system applies integration by parts as the rewrite rule

$$\int u \, dv = uv - \int v \, du,$$

choosing $u = \ln x$, $dv = x^2 dx$, to derive $I = \left[\frac{x^3}{3}\ln x\right]_0^1 - \int_0^1 \frac{x^2}{3}\,dx = 0 - \frac{1}{9} = -\frac{1}{9}$ exactly. The result is not a number but a proof trace—a sequence of identities each justifiable from axioms.

Key theorem (Church–Rosser / confluence). A term-rewriting system $R$ is confluent if for all terms $t, u, v$ with $t \twostar{R} u$ and $t \twostar{R} v$, there exists $w$ such that $u \twostar{R} w$ and $v \twostar{R} w$. Confluence guarantees that the normal form of any expression is unique regardless of the reduction order chosen, making symbolic results reproducible and auditable—properties numerical methods cannot provide.
