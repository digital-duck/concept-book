Array

Array

Definition. An array is a finite, ordered collection of elements drawn from a set $S$, stored in contiguous memory locations and indexed by a set of integers. A one-dimensional array $A$ of length $n$ is formally a function

$$A : \{0, 1, \ldots, n-1\} \to S,$$

where $A[i]$ denotes the element at index $i$. The size of $A$ is $n$, and the element type is the type of every $s \in S$. Multi-dimensional arrays generalize this: a matrix $M \in S^{m \times n}$ is indexed by pairs $(i, j)$ with $0 \le i < m$ and $0 \le j < n$.

Memory layout. In row-major order (used by C and Python/NumPy), the element $A[i][j]$ of an $m \times n$ array is stored at byte offset

$$\text{offset}(i,j) = (i \cdot n + j) \cdot w,$$

where $w$ is the width in bytes of one element. This formula makes row traversal cache-friendly and column traversal potentially expensive—a distinction with significant performance consequences.

Worked example. Let $A = [3, 7, 2, 9, 4]$ be a zero-indexed integer array of length $5$ with $w = 4$ bytes, base address $\alpha = 1000$.

  • $A[0]$ is at address $1000 + 0 \cdot 4 = 1000$.
  • $A[3]$ is at address $1000 + 3 \cdot 4 = 1012$.
  • The sum $\sum_{i=0}^{4} A[i] = 3+7+2+9+4 = 25$ is computed in $\Theta(n)$ time by a single linear scan.

    • Key theorem (random-access guarantee). For any valid index $i$, the element $A[i]$ can be read or written in $O(1)$ time. This follows directly from the offset formula: the address is computed arithmetically without any traversal.

    Proof sketch. Address computation requires one multiplication and one addition—two operations independent of $n$—followed by a single memory dereference. Hence the time is constant regardless of $n$. $\blacksquare$

    This $O(1)$ random access distinguishes arrays from linked lists, which require $O(n)$ traversal to reach an arbitrary element.

    Randomness

    Randomness

    Definition. A random experiment is any process whose outcome cannot be predicted with certainty before it occurs, even when all prior conditions are fixed. Formally, a probability space is a triple $(\Omega, \mathcal{F}, P)$, where $\Omega$ is the sample space (the set of all possible outcomes), $\mathcal{F}$ is a $\sigma$-algebra of events (measurable subsets of $\Omega$), and $P : \mathcal{F} \to [0,1]$ is a probability measure satisfying (i) $P(\Omega) = 1$, (ii) $P(\emptyset) = 0$, and (iii) countable additivity: for pairwise disjoint events $A_1, A_2, \ldots$,

    $$P\!\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i).$$

    A random variable $X : \Omega \to \mathbb{R}$ is a measurable function that assigns a numerical value to each outcome.

    Worked Example. Roll a fair six-sided die. Here $\Omega = \{1, 2, 3, 4, 5, 6\}$, $\mathcal{F} = 2^{\Omega}$ (the power set), and $P(\{\omega\}) = \frac{1}{6}$ for each $\omega \in \Omega$. Define $X(\omega) = \omega$. The probability of rolling an even number is

    $$P(X \in \{2,4,6\}) = P(\{2\}) + P(\{4\}) + P(\{6\}) = \frac{1}{6} + \frac{1}{6} + \frac{1}{6} = \frac{1}{2}.$$

    The expected value of $X$ is

    $$\mathbb{E}[X] = \sum_{\omega=1}^{6} \omega \cdot \frac{1}{6} = \frac{21}{6} = 3.5,$$

    illustrating that $\mathbb{E}[X]$ need not itself be a possible outcome.

    Key Theorem (Law of Large Numbers). Let $X_1, X_2, \ldots$ be independent and identically distributed random variables with finite mean $\mu = \mathbb{E}[X_i]$. Then, for every $\varepsilon > 0$,

    $$P\!\left(\left|\frac{1}{n}\sum_{i=1}^{n} X_i - \mu\right| > \varepsilon\right) \to 0 \quad \text{as } n \to \infty.$$

    This result grounds the frequentist interpretation of probability: the long-run relative frequency of an event converges to its probability, transforming an abstract axiom into an empirically testable prediction.

    Random Sampling

    Random Sampling

    Definition. Given a finite population $\mathcal{U} = \{u_1, u_2, \ldots, u_N\}$ of size $N$, a simple random sample of size $n \leq N$ is a subset $S \subseteq \mathcal{U}$ selected such that every possible subset of size $n$ is equally likely. The probability of selecting any particular subset is

    $$P(S) = \frac{1}{\binom{N}{n}}.$$

    In sampling with replacement, each draw is independent and the population is not altered between draws; in sampling without replacement (the more common setting in surveys), each selected unit is removed before the next draw. The estimator of the population mean $\mu = \frac{1}{N}\sum_{i=1}^{N} u_i$ from a sample $\{x_1, \ldots, x_n\}$ is the sample mean $\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$.

    Worked Example. Suppose a class has $N = 6$ students with exam scores $\{55, 62, 70, 78, 85, 91\}$, so the population mean is $\mu = 441/6 = 73.5$. Draw a simple random sample without replacement of size $n = 3$. There are $\binom{6}{3} = 20$ equally likely samples. If the draw yields $\{62, 78, 85\}$, the estimate is $\bar{x} = 225/3 = 75$. Across all 20 possible samples, the average of $\bar{x}$ equals $\mu = 73.5$, confirming unbiasedness.

    Key Theorem (Unbiasedness and Variance of $\bar{x}$). Under simple random sampling without replacement,

    $$\mathbb{E}[\bar{x}] = \mu, \qquad \operatorname{Var}(\bar{x}) = \frac{\sigma^2}{n}\left(1 - \frac{n}{N}\right),$$

    where $\sigma^2 = \frac{1}{N}\sum_{i=1}^{N}(u_i - \mu)^2$ is the population variance. The factor $\left(1 - \frac{n}{N}\right)$ is the finite population correction (FPC); it approaches 1 as $N \to \infty$, recovering the familiar $\operatorname{Var}(\bar{x}) = \sigma^2/n$ for large populations. The proof uses indicator variables $I_i \in \{0,1\}$ denoting whether unit $i$ is selected, with $\mathbb{E}[I_i] = n/N$ and $\operatorname{Cov}(I_i, I_j) = -\frac{n(N-n)}{N^2(N-1)}$ for $i \neq j$, from which both results follow by linearity of expectation.

    Lab Exercise. Using Python's random.sample, draw 1,000 independent samples of size $n = 30$ from a population of $N = 200$ values drawn from $\mathcal{N}(50, 100)$. For each sample compute $\bar{x}$. Plot the histogram of sample means and verify empirically that $\operatorname{Var}(\bar{x}) \approx \frac{\sigma^2}{n}\!\left(1 - \frac{n}{N}\right)$. Compare the result to sampling with replacement and explain the role of the FPC.

    Payoff

    Definition. Random sampling is the process of selecting a subset $S = \{x_1, x_2, \ldots, x_n\}$ of size $n$ from a population $\mathcal{X}$ according to a probability measure $\mu$ on $\mathcal{X}$, such that each draw is independent and identically distributed (i.i.d.) from $\mu$. The central promise is tractability: when $|\mathcal{X}|$ is astronomically large or $\mathcal{X}$ is a continuous space, random sampling replaces exhaustive enumeration with a statistically rigorous, computationally feasible surrogate.

    This concept is the natural endpoint of the preceding material precisely because it operationalizes probability. Every definition of expectation, variance, and convergence studied earlier was stated in terms of a distribution $\mu$. Random sampling is the mechanism that makes those abstractions empirically accessible—it is how a distribution becomes data.

    Worked Example. Suppose $\mathcal{X} = [0,1]$ and you wish to estimate $I = \int_0^1 f(x)\,dx$ for $f(x) = \sqrt{1-x^2}$ (one quarter of the unit disk area, so $I = \pi/4$). Draw $n$ samples $x_i \sim \text{Uniform}(0,1)$ and compute the Monte Carlo estimator $\hat{I}_n = \frac{1}{n}\sum_{i=1}^n f(x_i)$. For $n = 10{,}000$, a typical run yields $\hat{I}_{10000} \approx 0.7854$, matching $\pi/4 \approx 0.7854$ to four significant figures.

    Key Theorem (Law of Large Numbers). If $x_1, x_2, \ldots$ are i.i.d. with $\mathbb{E}[f(x_i)] = \mu_f < \infty$, then

    $$\hat{I}_n = \frac{1}{n}\sum_{i=1}^n f(x_i) \xrightarrow{\text{a.s.}} \mu_f \quad \text{as } n \to \infty.$$

    Moreover, by the Central Limit Theorem, $\sqrt{n}(\hat{I}_n - \mu_f) \xrightarrow{d} \mathcal{N}(0, \sigma_f^2)$, yielding the canonical $O(n^{-1/2})$ convergence rate—independent of the dimension of $\mathcal{X}$. This dimension-independence is what makes random sampling indispensable in high-dimensional statistics, machine learning, and simulation.

    Lab Exercise. Implement the Monte Carlo $\pi$ estimator above. Vary $n \in \{10^2, 10^3, 10^4, 10^5\}$, record $|\hat{I}_n - \pi/4|$ for each, and plot the error on a log-log scale. Verify empirically that the slope converges to $-\frac{1}{2}$, confirming the $O(n^{-1/2})$ rate. Then replace $f$ with a function of your choosing over $[0,1]^5$ and observe that the same slope persists—demonstrating the curse-of-dimensionality immunity that makes this technique foundational across all quantitative disciplines.
