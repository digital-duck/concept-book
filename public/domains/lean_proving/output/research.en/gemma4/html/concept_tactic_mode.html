🍃 Tactic Mode

🍃 Tactic Mode

Definition (Tactic Mode). Let $\mathcal{P}$ be an interactive proof system defined over a logical calculus $\mathcal{L}$. A tactic mode $M \in \mathcal{M}$ is formally defined as a state transition function that modifies the available set of inference rules $\Gamma$ and the structure of the goal type $G$. This modification dictates that when the current proof object reaches this mode, subsequent computational steps are governed by an altered meta-theory. Specifically, if the system transitions from mode $M_1$ to $M_2$, the new context $(C', \mathcal{L}')$ must satisfy $\mathcal{L}' \subseteq \text{Intra}(M_2)$, where $\text{Intra}(M_2)$ represents the set of internal deduction rules permissible solely within that mode. Furthermore, we introduce the concept of an enabling primitive $\mathcal{E}$, which denotes a foundational computational resource required for $M$ to operate; thus, any transition must preserve the satisfaction of $\mathcal{E}: (C', \mathcal{L}') \models \text{Safe}(M_2)$.

Theorem (Equivalence of Modes). For any two tactic modes $M_A$ and $M_B$, if $M_A \vdash M_B$ denotes a valid transition from $M_A$ to $M_B$, then the computational completeness of the subsystem defined by $M_B$ must be strictly greater than or equal to that of $M_A$: $\text{Comp}(M_B) \ge \text{Comp}(M_A)$. Furthermore, assuming a finite and reducible system, there exists a unique minimal set of modes $\{M_{min}\}$ such that any proof can be constructed by traversing only these necessary modes.

Proof Sketch. The computational completeness $\text{Comp}(M)$ measures the expressive power required to deduce all consequences within mode $M$. Since a transition $M_A \to M_B$ implies that $M_B$'s rules are an extension of or specialized relative to $M_A$'s environment, and crucially maintains the satisfaction of $\mathcal{E}$, the deductive capacity cannot decrease. If $\text{Comp}(M_B) < \text{Comp}(M_A)$, there must exist a theorem provable in $M_A$ but unprovable in $M_B$, contradicting the assumption of a valid transition within an extended proof context (see [Hyland & Pitts, 1987] for related concepts in type theory). The uniqueness of $\{M_{min}\}$ relies on demonstrating that any mode not essential for maintaining $\text{Comp}$ relative to its neighbors introduces unnecessary computational overhead without increasing the overall deductive power.

Remark. This concept formalizes and generalizes the notion of restricted inference systems found in formal logic and proof mining. In functional settings, $M$ can be interpreted as a context-sensitive refinement of the underlying type system, effectively constraining the allowed functors or constructors to ensure properties such as termination or specific structural guarantees (e.g., transitioning from general lambda calculus deduction to linear logic deduction). The study of mode transitions, especially in relation to enabling primitives $\mathcal{E}$, is crucial for developing modular and highly optimized interactive theorem provers capable of managing computational resources precisely.
