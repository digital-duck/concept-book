🍃 Core Tactics

Definition. Let $\mathcal{T}$ be a set of available tactical primitives, and let $S$ be the state space of a system under consideration. A core tactic $\tau \in \mathcal{T}$ is defined as an optimal sequence of actions $A = \{a_1, a_2, \dots, a_k\}$ such that for any initial state $s_0$, applying $\tau$ maximizes the expected utility function $U(s_k | s_0) = E[u(s_k)]$ relative to all other sequences of actions starting from $s_0$. Formally, $\tau^* = \arg\max_{\tau'} U(\text{Outcome}(\tau') | s_{init})$.

Theorem. For a finite-state system with bounded reward functions and complete observability, the set of core tactics $\mathcal{T}_{core}$ constitutes a basis for optimal policy construction, meaning any globally optimal policy $\pi^*$ can be decomposed into a sequence of core tactics: $\pi^* = \langle \tau_1, \tau_2, \dots, \tau_m \rangle$.

Proof. We proceed by contradiction. Assume there exists an optimal policy $\pi^*$ that cannot be fully decomposed into the set of core tactics $\mathcal{T}_{core}$. This implies that at some state $s_i$, the decision made by $\pi^*$, say $a_{i+1}$, does not correspond to a locally maximizing action defined by any $\tau \in \mathcal{T}_{core}$. Since $\mathcal{T}_{core}$ is defined based on local maximization of expected utility, if $a_{i+1}$ deviates from the predicted optimal move, it must be the case that there exists an alternative core tactic $\tau'$ starting at $s_i$ such that $U(\text{Outcome}(\tau') | s_i) > U(\text{Outcome}(a_{i+1} \dots) | s_i)$. This contradicts the assumption that $\pi^*$ was globally optimal, as we have found a superior path $\tau'$. Therefore, every component of an optimal policy must align with core tactics.

Remark. The assumption of finite state space and boundedness is crucial; otherwise, determining $\mathcal{T}_{core}$ reduces to solving the general challenge of finding Nash Equilibria in stochastic games (cf. $[\text{Harsanyi}, 1967]$). Furthermore, this concept generalizes the notion of optimal pathfinding in Markov Decision Processes (MDPs) by explicitly structuring the solution space into fundamental, reusable tactical units $\tau$. The identification of these basis tactics is central to developing compositional AI architectures capable of hierarchical planning.
