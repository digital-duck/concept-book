🍃 Symptom

A symptom ($\mathcal{S}$) is formally defined as an observable or measurable manifestation that signals a deviation from a normative baseline state $\Omega$ within a biological system $B$. Operationally, we consider the symptom set $\mathcal{S} = \{s_1, s_2, \dots, s_k\}$, where each $s_i$ is an indicator variable derived from empirical observation or quantitative measurement. Mathematically, a symptom represents a necessary condition for diagnosing an underlying pathological state $P$, such that if $P$ is present, then $\mathcal{S}$ must exhibit certain characteristics; however, the mere presence of $\mathcal{S}$ does not guarantee $P$. This formulation shifts the focus from simple observation to logical inference.

Theorem (Necessity and Sufficiency): Let $P$ be a pathological state and $\mathcal{S}(P)$ be the set of symptoms characteristically associated with $P$. Then, $\text{Diagnosis}(P) \implies \text{Observe}(\mathcal{S}_{min}) \subseteq \mathcal{S}$, where $\mathcal{S}_{min}$ is the minimal necessary subset of observable symptoms. However, the converse implication, $\mathcal{S} \subseteq \text{Observe}(\mathcal{S})$, does not generally hold due to confounding variables or atypical presentations.

Proof: We assume $P$ is present. By definition, $P$ generates a set of physiological changes that manifest as symptoms $\mathcal{S}(P)$. Therefore, if $P$ truly exists, all defining characteristics $\mathcal{S}_{min}$ must be observable (i.e., the implication holds). The failure of the converse ($\mathcal{S} \not\implies P$) is demonstrated by counterexamples where a set of symptoms $\mathcal{S}'$ might arise from an entirely different pathology $P' \neq P$, or could result from non-pathological confounders (e.g., environmental factors, measurement error). Formal diagnostic reasoning must thus rely on probabilistic inference rather than simple set containment.

Remark: This framework connects directly to axiomatic differential diagnosis systems and Bayesian network models of disease prediction. The probability $P(P | \mathcal{S})$—the likelihood of pathology $P$ given the observed symptom set $\mathcal{S}$—is central, necessitating careful calibration using pre-existing epidemiological data (e.g., prevalence rates) to avoid Type I or Type II errors in diagnosis [1].
