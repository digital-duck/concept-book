🌱 Position

When you use a GPS device, you are calculating your position—a precise location relative to an established global origin. This concept is fundamental to describing motion and change in physics. If we ignore vertical movement for simplicity, the simplest model of position treats an object's location as a single coordinate value on a number line. Mathematically, the position ($\vec{x}$) is defined by this coordinate. It answers the question: "Where are you right now?" Understanding position requires establishing a reference point (the origin) and a directionality (a positive axis).

Definition: Position $\vec{x}$ is a vector quantity that specifies an object's location relative to a chosen origin within a defined coordinate system. In one dimension, it is represented by the coordinate $x$. If we move from an initial position $\vec{x}_i$ to a final position $\vec{x}_f$, we define the change in position ($\Delta \vec{x}$) as the difference between these two endpoints:

$$ \Delta \vec{x} = \vec{x}_f - \vec{x}_i $$

Worked Example: A particle starts at $x_i = 2.0$ meters on a track and moves to a final position of $x_f = 7.5$ meters. We calculate the displacement (change in position):

$$ \Delta x = 7.5 \text{ m} - 2.0 \text{ m} = 5.5 \text{ m} $$

The particle's change in position is $5.5$ meters, regardless of how complex its actual path was between these two points.

Why it’s true: The linearity of the coordinate system allows us to use simple subtraction. Position is inherently defined relative to a fixed origin (often set at $x=0$). By subtracting the initial location from the final location, we are mathematically isolating only the net change in coordinates—the displacement. This definition holds because all motion can be resolved along these perpendicular axes.
