Concept Book: Import

Payoff

Definition

The word import — from Latin importāre, "to carry in" — is the lexical culmination of every morphological principle developed in this book. Formally, let $\Sigma^$ denote the set of all strings over an alphabet, and let $M$ be the morpheme lexicon equipped with a concatenation operation $\oplus$. A morphological construction is a well-formed element of the free monoid $(\mathcal{M}, \oplus, \varepsilon)$, where $\mathcal{M} \subseteq \Sigma^$ is the set of licensed morpheme sequences. Import is the image of the composition

$$\text{import} = \textit{im-} \oplus \textit{port},$$

where $\textit{im-}$ is the nasal-assimilated allomorph of the prefix $\textit{in-}$ (direction: into), and $\textit{port}$ is the Latin root for "carry." Every analytic tool introduced previously — allomorphy, prefixation, root identity, and derivational productivity — converges on this single lexical node.

Worked Example

Consider the morphological family rooted at $\textit{port}$:

| Construction | Decomposition | Gloss | |---|---|---| | export | $\textit{ex-} \oplus \textit{port}$ | carry out | | transport | $\textit{trans-} \oplus \textit{port}$ | carry across | | portable | $\textit{port} \oplus \textit{-able}$ | able to be carried | | import | $\textit{im-} \oplus \textit{port}$ | carry into |

The prefix $\textit{in-}$ surfaces as $\textit{im-}$ before the bilabial $[p]$, illustrating place-assimilation: $\phi(\textit{in-}, \textit{port}) = \textit{im-}$. The Morphological Decoding Principle then predicts the meaning compositionally, requiring no lexical lookup beyond the two constituents.

Key Theorem

Theorem (Root Productivity). Let $R$ be a root morpheme and $P$ the set of licensed directional prefixes. Then the number of distinct words derivable from $R$ is bounded below by $|P|$, and each derivative is semantically transparent under the composition semantics of $M$. For the root $\textit{port}$, $|P| \geq 6$ (in-, ex-, trans-, re-, sup-, com-), confirming high productivity.

Proof sketch. Each $p_i \oplus R$ is a distinct string; semantic compositionality follows from the free-monoid interpretation of $\oplus$ and the assumption that morpheme meanings are independent tensors in a semantic space $\mathbb{R}^d$.

from sympy import symbols, Function

# Allomorphy rule: in- → im- before bilabial stops
import_form = assimilate(in_, port)

print(f"Semantic value: [carry] ∘ [into] = {import_form}")
