In-

In calculus, a fundamental concept is the notion of in-functions, which generalizes the idea of inside a function. An in-function is defined as a function that satisfies certain properties related to its input values.

Definition

Let $f: \mathbb{R} \rightarrow \mathbb{R}$ be a function. We say that $f$ is an in-function if for every $x \in \mathbb{R}$, and every sequence $\{x_n\}_{n=1}^\infty$ of distinct real numbers converging to $x$, we have \[ f(x_n) \rightarrow f(x). \] This definition implies that the value of the function is constant when approaching a point from any direction.

Worked Example

Consider the function $f: \mathbb{R} \rightarrow \mathbb{R}$ defined by $f(x) = |x|$. Let $\{x_n\}_{n=1}^\infty$ be a sequence of distinct real numbers converging to $0$. Then, we have \[ f(x_n) = |x_n| \rightarrow 0. \] However, if we consider the function $g: \mathbb{R} \rightarrow \mathbb{R}$ defined by $g(x) = -x$, then for any sequence $\{x_n\}_{n=1}^\infty$ converging to $-2$, we have \[ g(x_n) = -(x_n) \rightarrow 2. \] This example illustrates that the condition holds only when $f(x)$ is non-negative.

Key Theorem

If $f: [a, b] \rightarrow [c, d]$ is an in-function, then for every $x \in (a, b)$ and every sequence $\{x_n\}_{n=1}^\infty$ of distinct real numbers converging to $x$, we have \[ f(x) = f(\lim_{n\to\infty} x_n). \]

Lab Cell (SymPy)

from sympy import symbols, limit

# Define the function
def f(x):
    return abs(x)

# Evaluate the limit
print(limit(f(symbols('x')), x, 0))

# Alternative example: define another in-function
def g(x):
    return -x

print(limit(g(symbols('x')), x, -2))

In this lab cell, we utilize SymPy to verify the key theorem for an additional in-function, $g(x) = -x$.
