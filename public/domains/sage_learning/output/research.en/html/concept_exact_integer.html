🌱 Exact Integer

Definition. Let $R$ be an integral domain containing the ring of integers $\mathbb{Z}$. We define an element $\alpha \in R$ to be an exact integer if, relative to any basis $\{b_i\}_{i=1}^n$ of $R/\mathbb{Q}$ (assuming $R$ is contained in a number field extension), the coordinates of $\alpha$ when expressed in this basis are all integers. More formally, if $R \otimes_{\mathbb{Z}} \mathbb{Q} = K$ is a number field, and $\{b_i\}_{i=1}^n$ is an integral basis for $K$, then $\alpha$ is an exact integer if $\alpha = \sum_{i=1}^n z_i b_i$ where $z_i \in \mathbb{Z}$. If the domain $R$ is already defined as a ring of algebraic integers, then every element is trivially an exact integer relative to its own integral basis.

Theorem. Let $\mathcal{O}_K$ be the ring of algebraic integers of a number field $K$. An element $\alpha \in \mathcal{O}_K$ is an exact integer if and only if it belongs to the lattice generated by the power basis $\{1, \omega_2, \dots, \omega_n\}$ where $\{\omega_i\}$ forms an integral basis for $\mathcal{O}_K$.
