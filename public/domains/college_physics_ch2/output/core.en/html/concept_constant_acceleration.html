Concept Book: Constant Acceleration

Constant Acceleration

Drop a ball from your hand. It doesn't just move — it moves *faster* every instant, gaining roughly $9.8\;\text{m/s}$ of speed each second. That steady increase is constant acceleration: the velocity changes by the same amount in every equal time interval.

Definition. An object has *constant acceleration* $a$ when its velocity $v$ changes linearly with time:

$$a = \frac{\Delta v}{\Delta t} = \text{const.}$$

From this single condition, two kinematic equations follow. Starting from position $x_0$ with initial velocity $v_0$ at $t = 0$:

$$v(t) = v_0 + at$$ $$x(t) = x_0 + v_0\,t + \tfrac{1}{2}\,a\,t^2$$

Worked Example. A car accelerates from rest ($v_0 = 0$) at $a = 3\;\text{m/s}^2$. How far does it travel in $4\;\text{s}$?

$$x = 0 + 0\cdot4 + \tfrac{1}{2}(3)(4)^2 = \tfrac{1}{2}\cdot 3\cdot 16 = 24\;\text{m}$$

Its final speed is $v = 0 + 3(4) = 12\;\text{m/s}$.

Why it's true. If acceleration is constant, velocity is a linear function of time, so $v(t) = v_0 + at$. Position is the integral of velocity:

$$x(t) = x_0 + \int_0^t (v_0 + a\,t')\,dt' = x_0 + v_0\,t + \tfrac{1}{2}\,a\,t^2$$

The $\tfrac{1}{2}$ arises because the velocity contribution grows linearly — geometrically, it is the area of a triangle sitting on top of a rectangle in a $v$-vs-$t$ graph. This derivation requires only that $a$ does not change; the moment acceleration varies, these neat formulas no longer apply directly.

Practice Problem. A stone is thrown straight upward with $v_0 = 15\;\text{m/s}$. Taking $a = -9.8\;\text{m/s}^2$ (directed downward), find the time it takes to reach its highest point and the maximum height above the launch position.
