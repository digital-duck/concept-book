Domain

Imagine a vending machine: you press a button, and it dispenses a snack. But not every button works — some are empty, some are out of service. The set of buttons that *actually work* is the machine's "valid input set." In mathematics, we call this the domain.

Definition. Given a function $f : A \to B$, the domain of $f$ is the set $A$ — the complete collection of input values for which $f$ is defined and produces an output in $B$.

When a function is given by a formula without an explicit domain, we take the natural domain: the largest subset of $\mathbb{R}$ (or the relevant set) for which the formula is meaningful.

Worked Example. Find the natural domain of

$$f(x) = \frac{\sqrt{x - 3}}{x - 7}.$$

We need two conditions to hold simultaneously:

  • The square root requires $x - 3 \geq 0$, so $x \geq 3$.
  • The denominator cannot be zero, so $x \neq 7$.

    • Intersecting these constraints gives

    $$\text{dom}(f) = [3, 7) \cup (7, \infty).$$

    Why it's true. The square root $\sqrt{u}$ is only real-valued when $u \geq 0$, and division by zero is undefined in any field. Since a function must assign *exactly one* output to each input, any $x$ that violates either condition cannot belong to the domain — including it would leave $f(x)$ undefined or ambiguous.

    This is why specifying the domain is not a bureaucratic detail: it is part of the definition of the function itself. Two functions with the same formula but different domains are, formally, different functions.

    $$g(x) = \ln(x^2 - 4).$$
